---

## First-party engine

Yumma CSS now features a first-party engine. This internal change supports future projects like [Yumma CSS Play](https://play.yummacss.com) without impacting existing projects.

---

## Opacity color variants

You can now use color utility variants to apply opacity to specific utilities.

<Preview>
  <div className="d-g g-4 gtc-4">
    <div className="bg-indigo/5 d-16"></div>
    <div className="bg-indigo/10 d-16"></div>
    <div className="bg-indigo/25 d-16"></div>
    <div className="bg-indigo/50 d-16"></div>
  </div>
</Preview>

```html "bg-indigo/5" "bg-indigo/10" "bg-indigo/25" "bg-indigo/50"
<div class="d-g g-4 gtc-4">
  <div class="bg-indigo/5 d-16"></div>
  <div class="bg-indigo/10 d-16"></div>
  <div class="bg-indigo/25 d-16"></div>
  <div class="bg-indigo/50 d-16"></div>
</div>
```
