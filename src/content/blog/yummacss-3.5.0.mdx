## First-party engine

That's right, we decided to create our own first-party engine for Yumma CSS. This shouldn't have any impact on your projects, but it's a big deal for us internally.

This change will allow us to extend Yumma CSS to feature projects like [Yumma CSS Play](https://play.yummacss.com) and more.

---

## Opacity color variants

You can now use color utility variants to specifically apply opacity to single utilities.

<Preview>
  <div className="d-g g-4 gtc-4">
    <div className="bg-indigo/5 d-16"></div>
    <div className="bg-indigo/10 d-16"></div>
    <div className="bg-indigo/25 d-16"></div>
    <div className="bg-indigo/50 d-16"></div>
  </div>
</Preview>

```html "bg-indigo/5" "bg-indigo/10" "bg-indigo/25" "bg-indigo/50"
<div class="d-g g-4 gtc-4">
  <div class="bg-indigo/5 d-16"></div>
  <div class="bg-indigo/10 d-16"></div>

</div>
```
