## First-Party Engine

Yumma CSS now features a first-party engine. This internal change supports future projects like [Yumma CSS Play](https://play.yummacss.com) without impacting existing projects.

---

## Opacity Color Variants

You can now use color utility variants to apply opacity to specific utilities.
