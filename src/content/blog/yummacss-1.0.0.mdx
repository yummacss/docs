## All-New Utilities

Added new utilities to expand CSS property coverage.

| Utility | Property                       |
|---------|--------------------------------|
| `ai-fs` | `align-items: flex-start;`     |
| `ai-fe` | `align-items: flex-end;`       |
| `jc-e`  | `justify-content: end;`        |
| `jc-fs` | `justify-content: flex-start;` |
| `jc-fe` | `justify-content: flex-end;`   |
| `jc-l`  | `justify-content: left;`       |
| `jc-r`  | `justify-content: right;`      |

## Viewport Expansion

Added viewport support for **height** & **width** utilities.

```html "md:h-auto" "md:w-auto"
<div class="h-10 md:h-auto md:w-auto w-10 ...">...</div>
```

| Color  | Value     |

## Syntax Changes

Refined utility syntax for improved consistency.

```html del={1} ins={2}
<div class="dis-*"></div>
<div class="d-*"></div>
```

Relocated hover state prefixes to the start of utilities for better readability.

```html del={1} ins={2}
<div class="bg-h-blue"></div>
<div class="h:bg-blue"></div>
```
