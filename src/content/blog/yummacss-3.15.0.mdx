## Container Queries

Container Queries allow you to style elements based on the size of their parent container.

```html "@sm:w-full"
<div class="w-half @sm:w-full">Responsive based on container size!</div>
```

## Standalone Scale

Following our transition to standalone properties for `rotate`, we've now refactored `scale` to use the standalone property.

| Prefix | Property  |
| ------ | --------- |
| `s-`   | `scale`   |
| `sx-`  | `scale-x` |
| `sy-`  | `scale-y` |
| `sz-`  | `scale-z` |

## Refreshed Box Shadows

We've completely overhauled our box shadow system. Shadows are now split into **Outset** & **Inset** categories with refined values from `xs` to `3xl`.

```html "bs-o-md" "bs-i-sm"
<!-- Outset Shadow -->
<div class="bs-o-md …">…</div>

<!-- Inset Shadow -->
<div class="bs-i-sm …">…</div>
```
