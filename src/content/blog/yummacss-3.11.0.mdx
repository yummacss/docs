## New transition properties

This release focuses on expanding our transitions system with three new utility properties specifically designed for animating element dimensions.

You can now target them specifically for better performance and control.

### Targeted dimension transitions

We've added three new prefixes to the `tp-*` (`transition-property`) category:

| Prefix | Property                              |
|--------|---------------------------------------|
| `tp-h` | `transition-property: height;`        |
| `tp-w` | `transition-property: width;`         |
| `tp-d` | `transition-property: height, width;` |

These utilities are perfect for accordions, dropdowns, or any component that needs to smoothly change size.

<Preview>
  <div className="bg-indigo h-16 w-16 tp-d td-300 tdu-300 ttf-io h:h-32 h:w-32 d-f ai-c jc-c c-white"></div>
</Preview>

```html "tp-d" "td-300" "tdu-300" "ttf-io"
<!-- Height and width transition -->
<div class="h-16 w-16 tp-d td-300 tdu-300 ttf-io h:h-32 h:w-32 ..."></div>
```

If you only need to animate one axis, you can use the more specific utilities:

```html "tp-h" "tp-w"
<!-- Only height -->
<div class="tp-h h:h-64 ..."></div>

<!-- Only width -->
<div class="tp-w h:w-64 ..."></div>
```
