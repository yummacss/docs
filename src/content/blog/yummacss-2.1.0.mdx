## Base styles

So buttons can act more like buttons.

```scss mark={2} title="_base.scss"
button {
  cursor: pointer;
}
```

New `font-size: inherit` and `font-weight: inherit` default rules.

```scss mark={8-9} title="_base.scss"
h1,
h2,
h3,
h4,
h5,
h6,
p {
  font-size: inherit;
  font-weight: inherit;
  overflow-wrap: break-word;
}
```
