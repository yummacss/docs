## All-New Utilities

Continuous utility expansion remains a priority. This major release introduces several new tools for design control.

### Accent Color

```html "ac-(value)"
<input class="ac-red" type="checkbox" checked />
<input class="ac-orange" type="checkbox" checked />
<input class="ac-yellow" type="checkbox" checked />
```

## Base Styles

Disable default base styles by importing `yumma-core.css` instead of `yumma.css`.

#### Standard CSS

```html
<link
  href="https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@2.0.0/dist/yumma-core.css"
  rel="stylesheet"
  crossorigin="anonymous"
/>
```

#### Minified CSS

```html
<link
  href="https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@2.0.0/dist/yumma-core.min.css"
  rel="stylesheet"
  crossorigin="anonymous"
/>
```

## Media Queries

Media query support is now available across nearly all Yumma CSS utilities for better responsiveness.

```html "sm:gtc-2" "md:gtc-3" "lg:gtc-4"
<div class="mx-auto p-4">
  <div class="d-g gtc-1 g-4 sm:gtc-2 md:gtc-3 lg:gtc-4">
    <div class="…">…</div>
    <div class="…">…</div>
    <div class="…">…</div>
  </div>
</div>
```

## Container Rules

Introduced the `cnt` class for containers with built-in media query rules for responsiveness.

<Tabs defaultValue="v2.0">
    <TabsList>
        <TabsTab value="v2.0">v2.0</TabsTab>
        <TabsTab value="v1.2">v1.2</TabsTab>
    </TabsList>

  <TabsPanel value="v2.0">
    ```scss
    .cnt {
      width: 100%;
      @each $bp, $bp_value in $yma-breakpoints {
        @include breakpoint($bp_value) {
          max-width: $bp_value;
        }
      }
    }
    ```
  </TabsPanel>

  <TabsPanel value="v1.2">
    ```scss
    .cnn {
      width: 100%;
      max-width: 1200px;
      margin: 0 auto;
      padding: 0 20px;
      box-sizing: border-box;
    }
    ```
  </TabsPanel>
</Tabs>
