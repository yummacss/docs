## New Pseudo Class Variants

Yumma CSS v3.1.0 adds support for focus & active pseudo-class variants compatible with all base utilities.

### Focus Variants

Apply styles to elements that receive focus using the `f:` prefix, improving form & button accessibility.

<Preview className="br-sm">
  <button className="w-14 h-14 px-4 py-2 bg-indigo h:bg-indigo-7 f:oc-indigo-5 f:oo-2 f:os-s f:ow-2"></button>
</Preview>

```html "f:oc-indigo-5 f:oo-2 f:os-s f:ow-2"
<button
  class="w-14 h-14 px-4 py-2 bg-indigo h:bg-indigo-7 f:oc-indigo-5 f:oo-2 f:os-s f:ow-2"
></button>
```

### Active Variants

The `a:` prefix styles elements during interaction, perfect for button feedback & touch interactions.

<Preview className="br-sm">
  <div className="w-14 h-14 px-4 py-2 bg-indigo a:bg-indigo-9"></div>
</Preview>

```html "a:bg-indigo-9"
<div class="w-14 h-14 px-4 py-2 bg-indigo a:bg-indigo-9"></div>
```
