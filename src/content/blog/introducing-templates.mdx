## What's included?

Every template includes essential features to get you started:

- **Multiple pages** - Pre-built pages like Home, About, Services, & more.
- **Responsive design** - Fully responsive layouts for all devices.
- **Clean code** - Built with TypeScript & best practices.
- **Lifetime updates** - One-time purchase with perpetual updates.

---

## Tech stack

Templates are built with modern technologies:

- **Yumma CSS** - Styling
- **Next.js** - Server-side rendering
- **React** - Frontend framework
- **pnpm** - Package management
- **TypeScript** - Type safety

Templates are built on [Yumma CSS](/docs) for easy customization.
