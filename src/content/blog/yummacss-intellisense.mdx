Building with abbreviated utility classes is fast, but it shouldn't mean guessing which one to use. [Yumma CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=yumma-css.yummacss-intellisense) brings the power of the framework directly into your editor, giving you instant feedback and suggestions as you build.

## Key Features

### IntelliSense

Deeply integrated auto-completion for Yumma CSS's abbreviated utility classes.

![IntelliSense auto-completion](/blog/yummacss-intellisense-completion.png)

### Hover Previews

Instant transparency. Hover over any class to see the exact CSS declaration it generates.

![Hover previews](/blog/yummacss-intellisense-hover.png)

### Smart Diagnostics

Real-time conflict detection that identifies overlapping styles as you type.

![Smart diagnostics](/blog/yummacss-intellisense-diagnostics.png)

### Quick Fixes

One-click resolutions to keep your class lists clean and conflict-free.

![Quick fixes](/blog/yummacss-intellisense-quick-fixes.png)
