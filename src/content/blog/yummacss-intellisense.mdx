Abbreviated utility classes are efficient, but shouldn't require guesswork. [Yumma CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=yumma-css.yummacss-intellisense) provides suggestions, previews, & diagnostics directly in the editor.

## Key Features

### Intellisense

Context-aware auto-completion for Yumma CSS abbreviated utility classes.

![IntelliSense auto-completion](/blog/yummacss-intellisense-completion.png)

### Hover Previews

Hover over any class to see the generated CSS declaration.

![Hover previews](/blog/yummacss-intellisense-hover.png)

### Smart Diagnostics

Real-time conflict detection identifies overlapping styles as you type.

![Smart diagnostics](/blog/yummacss-intellisense-diagnostics.png)

### Quick Fixes

Resolve styles & keep class lists clean with automated resolutions.
