## What is Yumma UI?

Yumma UI is a collection of over 100+ fully responsive, accessible, and customizable UI components, designed with Yumma CSS ready to work with your React or Next.js projects.

---

## Get Started

Getting started with Yumma UI is easy. You just need to have Yumma CSS installed in your project.

<Stepper>
  <Step title="Install Yumma CSS">
    First, ensure you have Yumma CSS set up.

    ```bash
    npm install yummacss
    ```
  </Step>
  <Step title="Browse Components">
    Visit the [UI documentation](/ui) to browse the available components.
  </Step>
  <Step title="Copy and Paste">
    You can use the component code as it is or customize it to match your brand.
  </Step>
</Stepper>

## How can I support Yumma UI?

We are excited to announce that **Yumma UI Templates** will soon be available on Yumma UI.

![Lightbox Template](/templates/lightbox.png)

These highly customizable, production-ready React and Next.js templates are available for a one-time purchase.

By purchasing a template, you're supporting the project, getting a head start on your next big idea, and helping us keep Yumma UI free.

Your purchase helps us continue developing Yumma UI and allows us to invest more time into improving it. Your support is greatly appreciated!
