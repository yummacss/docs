## Box model utility changes

Range **height**, **margin**, **padding**, & **width** utilities from 0 to 100.

```html
<div class="h-[0/100]">...</div>
<div class="p-[0/100]">...</div>
<div class="m-[0/100]">...</div>
<div class="w-[0/100]">...</div>
```

---

## Upgrade

Get the latest version of Yumma CSS from npm or pnpm:

```bash
pnpm add yummacss@latest
```
