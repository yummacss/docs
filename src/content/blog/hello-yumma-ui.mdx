## What is Yumma UI?

Building polished UIs with utility CSS often involves rebuilding the same buttons, alerts, & layouts. Yumma UI provides production-ready blocks so you can start from a real interface instead of an empty file.

## Get started

Getting started with Yumma UI requires [Yumma CSS](/docs).

<Stepper>
  <Step title="Install Yumma CSS">
    Ensure Yumma CSS is set up in your project.

    ```bash
    pnpm add yummacss
    ```
  </Step>
  <Step title="Browse components">
    Visit the [UI documentation](/ui) to browse the available components.
  </Step>
  <Step title="Copy & paste">
    Use component code as is or customize it for your brand.
  </Step>
</Stepper>
