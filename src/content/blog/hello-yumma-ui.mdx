## What Is Yumma UI?

Building polished UIs with utility CSS means rebuilding the same buttons, alerts, & layouts. Yumma UI provides production-ready blocks so you start from a real interface instead of an empty file.

---

## Get Started

Getting started with Yumma UI requires [Yumma CSS](/docs).

<Stepper>
  <Step title="Install Yumma CSS">
    Set up Yumma CSS in your project. ```bash pnpm add yummacss ```
  </Step>
  <Step title="Browse Components">
    Visit the [UI documentation](/ui) to browse available components.
  </Step>
  <Step title="Copy & Paste">
    Use component code as is or customize it for your brand.
  </Step>
</Stepper>
