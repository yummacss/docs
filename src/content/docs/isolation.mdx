## Usage

### Auto

Allow the element to use the default isolation behavior.

```html "is-auto"
<div class="is-auto ..."></div>
```

### Isolate

Create a new stacking context for the element.

```html "is-i"
<div class="is-i ..."></div>
```

Use breakpoint prefixes to target different screen sizes. Styles apply from the breakpoint & up.
