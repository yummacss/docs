## Outside

> Initial value

The list item markers will be placed outside the list item's content area, allowing the text to align normally.

<Preview>
  <ul className="lsp-o lst-d">
    <li className="c-slate">Butter 🧈</li>
    <li className="c-slate">Egg 🥚</li>
    <li className="c-slate">Milk 🥛</li>
  </ul>
</Preview>

## Inside

The list item markers will be placed inside the list item's content area, affecting the text alignment.

<Preview>
  <ul className="lsp-i lst-d">
    <li className="c-slate">Butter 🧈</li>
    <li className="c-slate">Egg 🥚</li>
    <li className="c-slate">Milk 🥛</li>
  </ul>
</Preview>

```html "lsp-i"
<ul class="lsp-i lst-d">
  <li class="c-slate">Butter 🧈</li>
  <li class="c-slate">Egg 🥚</li>
  <li class="c-slate">Milk 🥛</li>
</ul>
```
