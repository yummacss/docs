## None

> Initial value

Removes all default styling from the element.

```html live "a-none"
<button class="a-none px-5 py-1">Send</button>
```

## Auto

Allows the element to use the default styling provided by the browser.

```html live "a-auto"
<button class="a-auto px-5 py-1">Send</button>
```
