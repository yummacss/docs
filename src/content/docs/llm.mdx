## Using Yumma CSS with LLMs

If you ever asked an LLM to design a website with Yumma CSS, chances are it will use incorrect syntax, mix it with other frameworks, or invent non-existent class names.

This is because LLMs don't understand the syntax of Yumma CSS, and they might generate something that doesn't exist or is not valid.

### Why this happens?

LLMs need proper context in order to give you an accurate response, and since Yumma CSS is not a well-known framework, LLMs might not understand it.
