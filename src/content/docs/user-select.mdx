## Usage

### Auto

Allow the browser to determine the default highlighting behavior.

<Hint icon="cursor_text">Try selecting the sentence to see how the text highlighting works.</Hint>

<Preview>
  <div className="fs-lg c-slate us-auto">Sphinx of black quartz, judge my vow.</div>
</Preview>

```html "us-auto"
<p class="fs-lg c-slate us-auto">Sphinx of black quartz, judge my vow.</p>
```

### All

Allow the user to select all text within the element.

<Preview>
  <div className="fs-lg c-slate us-a">Sphinx of black quartz, judge my vow.</div>
</Preview>

```html "us-a"
<p class="fs-lg c-slate us-a">Sphinx of black quartz, judge my vow.</p>
```

### None

Prevent the user from selecting any text within the element.

<Preview>
  <div className="fs-lg c-slate us-none">Sphinx of black quartz, judge my vow.</div>
</Preview>

```html "us-none"
<p class="fs-lg c-slate us-none">Sphinx of black quartz, judge my vow.</p>
```

### Text

Allow the user to select text within the element.

<Preview>
  <div className="fs-lg c-slate us-t">Sphinx of black quartz, judge my vow.</div>
</Preview>

```html "us-t"
<p class="fs-lg c-slate us-t">Sphinx of black quartz, judge my vow.</p>
```
