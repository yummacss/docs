### Max Dimension

Control an element's maximum width & height simultaneously.

<Reference category="boxModel" name="max-dimension" />

The `max-d-64` utility limits an element's size to `16 rem` in both dimensions.

<Hint icon="size">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Hint>

<Preview>
  <div className="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">64</div>
</Preview>

```html "max-d-64"
<div class="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">64</div>
```

### Min Dimension

Control an element's minimum width & height simultaneously.

<Reference category="boxModel" name="min-dimension" />

The `min-d-32` utility ensures an element's size is at least `8 rem` in both dimensions.

<Hint icon="size">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Hint>

<Preview>
  <div className="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">32</div>
</Preview>

```html "min-d-32"
<div class="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">32</div>
```
