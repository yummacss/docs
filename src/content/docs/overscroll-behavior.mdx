## Usage

### Auto

Use the `ob-a` utility to allow scroll overflow behavior to propagate to parent elements.

```html "ob-a"
<div class="ob-a ...">
  ...
</div>
```

### Contain

Use the `ob-c` utility to prevent scroll overflow behavior from propagating to parent elements, but allow scroll bounce effects within the element.

```html "ob-c"
<div class="ob-c ...">
  ...
</div>
```

### None

Use the `ob-n` utility to prevent scroll overflow behavior from propagating to parent elements & disable scroll bounce effects.

```html "ob-n"
<div class="ob-n ...">
  ...
</div>
```
