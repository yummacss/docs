## Usage

### None

Prevent an element from being resized.

<Hint icon="size">Try resizing the text area to see how it changes in size.</Hint>

<Preview>
  <div className="d-f fd-c">
    <label className="mb-2 c-slate">Write a message:</label>
    <textarea className="px-3 py-2 bc-silver-4 bw-1 fs-md r-none f:oc-silver-1 f:os-s f:ow-2" rows={4}></textarea>
  </div>
</Preview>

```html "r-none"
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>
  <textarea class="px-3 py-2 bc-silver-4 bw-1 fs-md r-none f:oc-silver-1 f:os-s f:ow-2" rows="4"></textarea>
</div>
```

  </div>
</Preview>

```html "r-b"
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>

</div>
```
