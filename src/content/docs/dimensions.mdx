## Max Dimension

Controls the maximum height and max width of an element at once.

<Reference category="boxModel" name="max-dimension" />

The `max-d-64` utility ensures the element does not exceed a size of `16rem` in both height and width.

<Note icon="size">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Note>

<Preview>
  <div className="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">64</div>
</Preview>

```html "max-d-64"
<div class="o-auto ai-c bg-indigo d-16 d-f jc-c max-d-64 min-d-32 p-6 r-b c-white">64</div>
```

## Min Dimension

Controls the minimum height and min width of an element at once.

<Reference category="boxModel" name="min-dimension" />

The `min-d-32` utility ensures the element does not shrink below a size of `16rem` in both height and width.

<Note icon="size">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Note>
