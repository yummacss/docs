## Inline

> Initial value

Makes the element an inline element, allowing it to sit within a line of text without breaking the flow.

```html live "d-i" layout="/src/layouts/inline.astro"
<div class="bg-indigo d-i p-4 rad-1 tc-white">A</div>
<div class="bg-indigo d-i p-4 rad-1 tc-white">B</div>
<div class="bg-indigo d-i p-4 rad-1 tc-white">C</div>
```

## Block

Makes the element a block-level element, causing it to take up the full width available and start on a new line.

```html live "d-b" layout="/src/layouts/inline.astro"
<div class="d-b ta-c tc-white">
  <div class="bg-indigo mt-4 p-4 rad-1">A</div>
  <div class="bg-indigo mt-4 p-4 rad-1">B</div>
  <div class="bg-indigo mt-4 p-4 rad-1">C</div>
</div>
```

## Flex

Makes the element a flex container, enabling the use of flexbox layout properties for its children.

```html live "d-f" layout="/src/layouts/inline.astro"
<div class="d-f g-4 ta-c tc-white">
  <div class="ai-c bg-indigo d-b d-g d-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-b d-g d-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-b d-g d-14 jc-c p-4 rad-1">C</div>
</div>
```

## Flow Root

Creates a new block formatting context, allowing for better control of floated children.

```html live "d-fr" layout="/src/layouts/inline.astro"
<div class="d-fr ta-c tc-white">
  <div class="ai-c bg-indigo d-f d-18 flo-l jc-c p-a rad-1 zi-10">A</div>
  <div class="h-18"></div>
</div>
```

## Grid

Makes the element a grid container, enabling the use of grid layout properties for its children.

```html live "d-g" layout="/src/layouts/inline.astro"
<div class="d-g g-4 gtc-2 ta-c tc-white">
  <div class="bg-indigo p-4 rad-1">A</div>
  <div class="bg-indigo p-4 rad-1">B</div>
  <div class="bg-indigo p-4 rad-1">C</div>
</div>
```

## Inline Flex

Makes the element an inline flex container, allowing it to sit within a line of text while enabling flexbox layout properties for its children.

```html live "d-if" layout="/src/layouts/inline.astro"
<div class="d-if g-4 ta-c tc-white">
  <div class="bg-indigo d-ib p-4 rad-1 w-32">A</div>
  <div class="bg-indigo d-ib p-4 rad-1 w-32">B</div>
  <div class="bg-indigo d-ib p-4 rad-1 w-32">C</div>
</div>
```

## Inline Block

Makes the element an inline-block element, allowing it to sit within a line of text while also respecting width and height properties.

```html live "d-ib" layout="/src/layouts/inline.astro"
<div class="ta-c tc-white">
  <span class="bg-indigo d-ib ml-4 p-4 rad-1 w-32">A</span>
  <span class="bg-indigo d-ib ml-4 p-4 rad-1 w-32">B</span>
  <span class="bg-indigo d-ib ml-4 p-4 rad-1 w-32">C</span>
</div>
```

## Inline Grid

Makes the element an inline grid container, allowing it to sit within a line of text while enabling grid layout properties for its children.

```html live "d-ig"
<div class="d-ig gtc-2 g-4 ta-c ta-c tc-white">
  <div class="bg-indigo d-ib p-4 rad-1 w-32">A</div>
  <div class="bg-indigo d-ib p-4 rad-1 w-32">B</div>
  <div class="bg-indigo d-ib p-4 rad-1 w-32">C</div>
</div>
```

## Inline Table

Makes the element an inline table, allowing it to sit within a line of text while behaving like a table element.

```html "d-it" layout="/src/layouts/inline.astro"
<div class="d-it ..."></div>
```

## Table

Makes the element a block-level table, allowing it to contain table rows and cells.

```html "d-t" layout="/src/layouts/inline.astro"
<div class="d-t ..."></div>
```

## None

Hides the element, preventing it from being displayed in the layout.

```html live "d-none" layout="/src/layouts/inline.astro"
<div class="d-g rg-4 ta-c tc-white">
  <div class="bg-indigo d-none p-4 rad-1">A</div>
  <div class="bg-indigo p-4 rad-1">B</div>
  <div class="bg-indigo p-4 rad-1">C</div>
</div>
```
