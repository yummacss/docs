## Usage

### Auto

Align the item based on the container's default alignment properties.

<Preview variant="inline">
  <div className="d-g g-4 gtc-3 c-white" id="area">
    <div className="ai-c bg-indigo-8 d-f d-14 jc-c c-indigo-5">A</div>
    <div className="ai-c bg-indigo d-f d-14 jc-c js-auto c-white">B</div>
    <div className="ai-c bg-indigo-8 d-f d-14 jc-c c-indigo-5">C</div>
  </div>
</Preview>

```html "js-auto"
<div class="d-g g-4 gtc-3 c-white">
  <div class="ai-c bg-indigo-8 d-f d-14 jc-c c-indigo-5">A</div>
  <div class="ai-c bg-indigo d-f d-14 jc-c js-auto c-white">B</div>
  <div class="ai-c bg-indigo-8 d-f d-14 jc-c c-indigo-5">C</div>
</div>
```
