## Usage

### Separate

Table cell borders remain distinct, allowing for spacing.

<Preview>
  <table className="w-full bc-gray c-gray bw-1 bc-s ta-c">
    <tbody>
      <tr>
        <td className="p-2 bc-gray bw-1">A</td>
        <td className="p-2 bc-gray bw-1">B</td>
      </tr>
      <tr>
        <td className="p-2 bc-gray bw-1">C</td>
        <td className="p-2 bc-gray bw-1">D</td>
      </tr>
    </tbody>
  </table>
</Preview>
