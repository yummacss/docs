## Usage

### None

Allow the element to be positioned normally without clearing floats.

<Preview variant="inline">
  <div className="bg-indigo d-32 fl-l"></div>
  <div className="cl-none p-6" id="area"></div>
</Preview>

```html "cl-none"
<div class="bg-indigo d-32 fl-l"></div>
<div class="cl-none p-6"></div>
```

### Both

Move the element below floating elements on both sides.

<Preview variant="inline">
  <div className="bg-indigo d-32 fl-l"></div>
  <div className="bg-indigo d-32 fl-r"></div>
  <div className="cl-b p-6" id="area"></div>
</Preview>

```html "cl-b"
<div class="bg-indigo d-32 fl-l"></div>
<div class="bg-indigo d-32 fl-r"></div>
<div class="cl-b p-6"></div>
```
