## Legacy Versions

In this guide, **legacy** refers to Yumma CSS v2.1 & earlier versions. These older systems are no longer actively developed.

<Stepper>
  <Step title="Update Yumma CSS">
    Remove the legacy `yummacss` package & install the latest version. Install it as a dev dependency (`-D`) to keep your production bundle lean.

    ```bash
    pnpm up yummacss@latest
    ```

  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash
    pnpx yummacss init
    ```

  </Step>

  <Step title="Delete old imports">
    Remove the legacy `@import` statements from your CSS files.

    ```css title="styles.css" del={1-2}
    @import "yummacss/dist/yumma.min.css";
    @import "yummacss/dist/yumma.css";
    ```

  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the new `yumma.config.js` file.

    ```js title="yumma.config.js" mark={4}
    export default {
      source: ["./src/**/*.html"],
      output: "./src/styles.css",
      normalize: true,
    };
    ```

  </Step>

  <Step title="Build styles">
    Start generating your CSS with the [`build`](/docs/configuration/#build) or [`watch`](/docs/configuration/#watch) command.

    ```bash
    pnpx yummacss build
    ```

  </Step>
</Stepper>

## Dependency Changes

Yumma CSS now functions as a CLI tool. The `@yummacss/api` package is now `@yummacss/core`. Update your imports accordingly.

```bash
pnpm remove @yummacss/api
pnpm add @yummacss/core
```

## Utility Changes

Utilities renamed in the current version:

| Property             | Legacy (v2)               | Latest (v3.21.0)                |
| -------------------- | ------------------------- | ------------------------------- |
| Border Radius        | `rad-(value)`             | `br-(value)`                    |
| Border Radius (Pill) | `rad-9`                   | `br-pill`                       |
| Border Style         | `b-(value)`               | `bs-(value)`                    |
| Border Width         | `b-(value)`               | `bw-(value)`                    |
| Box Shadow           | `bs-(value)`              | `bs-o-(value)` & `bs-i-(value)` |
| Color (Text)         | `tc-(value)`              | `c-(value)`                     |
| Color (Lead)         | `lead`                    | `slate`                         |
| Color Variations     | `l-(value)` & `d-(value)` | Removed                         |
| Columns              | `cols-(value)`            | `c-(value)`                     |
| Direction (Bottom)   | `dir-b-(value)`           | `b-(value)`                     |
| Direction (Inset)    | `dir-i-(value)`           | `i-(value)`                     |
| Direction (Left)     | `dir-l-(value)`           | `l-(value)`                     |
| Direction (Right)    | `dir-r-(value)`           | `r-(value)`                     |
| Direction (Top)      | `dir-t-(value)`           | `t-(value)`                     |
| Float                | `flo-(value)`             | `fl-(value)`                    |
| Font Family (Serif)  | `ff-c`                    | `ff-s`                          |
| Font Size (1rem)     | `fs-b`                    | `fs-md`                         |
| Object Position      | `op-left`                 | `op-l`                          |
| Overflow             | `ovf-(value)`             | `o-(value)`                     |
| Viewport Units       | `*-1/1`                   | `*-dvh`                         |

## Removed Utilities

The following utilities were completely removed to reduce bloat & improve consistency.

| Category         | Legacy (v2)                                           | Replacement                                       |
| ---------------- | ----------------------------------------------------- | ------------------------------------------------- |
| Dimension        | `dim-(value)` / `max-dim-(value)` / `min-dim-(value)` | Use `w-(value)` & `h-(value)` together            |
| Color Variations | `l-(value)` & `d-(value)` (e.g., `l-red-5`)           | Use standard numbers (e.g., `red-3`)              |
| Spacing X/Y      | `sx-(value)` & `sy-(value)`                           | Use gap (`g-(value)`, `cg-(value)`, `rg-(value)`) |
| Container        | `cnt`                                                 | Removed                                           |
| Inset            | `ins`                                                 | Use `d-f ai-c jc-c`                               |

### Color Utility Changes

Remove light (`l-`) & dark (`d-`) character prefixes across all color utilities. The `lead` color is now `slate`. Map the legacy 1-6 scale to the new 1-12 scale.

```html del={1} ins={2} title="index.html"
<button class="bg-l-lead-6 h:bg-d-lead-1">Button</button>
<button class="bg-slate-1 h:bg-slate-7">Button</button>
```

| Legacy         | Latest        |
| -------------- | ------------- |
| `*-l-indigo-6` | `*-indigo-1`  |
| `*-l-indigo-5` | `*-indigo-2`  |
| `*-l-indigo-4` | `*-indigo-3`  |
| `*-l-indigo-3` | `*-indigo-4`  |
| `*-l-indigo-2` | `*-indigo-5`  |
| `*-l-indigo-1` | `*-indigo-6`  |
| `*-d-indigo-1` | `*-indigo-7`  |
| `*-d-indigo-2` | `*-indigo-8`  |
| `*-d-indigo-3` | `*-indigo-9`  |
| `*-d-indigo-4` | `*-indigo-10` |
| `*-d-indigo-5` | `*-indigo-11` |
| `*-d-indigo-6` | `*-indigo-12` |

### Layout Direction

Remove the `dir-` prefix for layout utilities.

```html del={1-5} ins={6-10} title="index.html"
<div class="dir-b-4">...</div>
<div class="dir-i-2">...</div>
<div class="dir-l-4">...</div>
<div class="dir-r-2">...</div>
<div class="dir-t-4">...</div>
<div class="b-4">...</div>
<div class="i-2">...</div>
<div class="l-4">...</div>
<div class="r-2">...</div>
<div class="t-4">...</div>
```

### Dynamic Viewport

Replace 1/1 dynamic viewport utilities with `dvh` & `dvw`.

```html del={1-2} ins={3-4} title="index.html"
<div class="w-1/1 h-1/1">...</div>
<div class="h-1/1">...</div>
<div class="w-dvw h-dvh">...</div>
<div class="h-dvh">...</div>
```

### Typography

Use shorter prefixes for font utilities. Replace the `fs-b` class with `fs-md`.

```html del={1-2} ins={3-4} title="index.html"
<div class="fs-b">...</div>
<div class="ff-c">...</div>
<div class="fs-md">...</div>
<div class="ff-s">...</div>
```

### Borders

Prevent conflicts with unique prefixes for border width (`bw-(value)`) & border style (`bs-(value)`). Replace border radius (`rad-(value)`) with `br-(value)`.

```html del={1} ins={2} title="index.html"
<div class="b-solid b-2 rad-9">...</div>
<div class="br-pill bs-s bw-2">...</div>
```

### Box Shadows

Apply outset (`bs-o-`) & inset (`bs-i-`) prefixes for box shadow utilities.

```html del={1} ins={2} title="index.html"
<div class="bs-md">...</div>
<div class="bs-o-md">...</div>
```

## Minify Option

The `minify` build option is removed. Choose modern tools like Vite to automate CSS minification instead.

## Base Styles

Modern versions include standardized base styles for consistency. Disable base styles in `yumma.config.js` if you prefer your own reset.

```js mark={2} title="yumma.config.js"
export default {
  normalize: false,
};
```
