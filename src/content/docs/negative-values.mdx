## Use Negative Values

Yumma CSS supports negative values for specific utilities like margins, positioning & letter spacing. Use the `{prefix}--{value}` syntax to apply a negative value.

## Margin

Use negative margins to overlap elements or create unique layouts. Negative margins are available for all margin utilities (`m-`, `mt-`, `mr-`, `mb-`, `ml-`, `mx-` & `my-`).

  <div className="d-f jc-c c-white fs-sm">

    <div className="d-f ai-c jc-c ml--4 w-14 h-14 bg-indigo bc-indigo-2/50 br-pill bw-2 z-2">B</div>

```html "ml--4"
<div class="d-f jc-c c-white fs-sm">
  <div class="d-f ai-c jc-c w-14 h-14 bg-indigo br-pill bw-2 z-1">A</div>
  <div class="d-f ai-c jc-c ml--4 w-14 h-14 bg-indigo br-pill bw-2 z-2">B</div>
  <div class="d-f ai-c jc-c ml--4 w-14 h-14 bg-indigo br-pill bw-2 z-3">C</div>
</div>
```

## Positioning

Apply negative offsets to position elements outside their parents or create overlapping effects. This works with `t-`, `r-`, `b-`, `l-` & `i-` utilities.

<Preview>
  <div className="p-r w-40 h-40 bg-silver-2 br-2">
    <div className="p-a t--4 l--4 w-12 h-12 bg-indigo br-1"></div>
  </div>
</Preview>

```html "t--4" "l--4"
<div class="p-r w-40 h-40 bg-silver-2 br-2">
  <div class="p-a t--4 l--4 w-12 h-12 bg-indigo br-1"></div>
</div>
```

## Z-Index

Negative z-index values allow you to place elements behind their siblings or parents. Use `zi--{value}` to target negative stack levels.

<Preview>
  <div className="d-f p-r ai-c jc-c w-full h-32 bg-silver-1 br-2">
    <div className="p-a zi--10 w-24 h-24 bg-indigo/20 br-pill"></div>
    <span className="c-slate fw-600">Background Layer</span>
  </div>
</Preview>

```html "zi--10"
<div class="d-f p-r ai-c jc-c w-full h-32 bg-silver-1 br-2">
  <div class="p-a zi--10 w-24 h-24 bg-indigo/20 br-pill"></div>
  <span>Background Layer</span>
</div>
```

## Letter Spacing

Utilities like `ls-1` (which is `-0.05 em`) already represent negative values. Using the negative syntax `ls--1` will toggle the sign, resulting in a positive value.

<Hint icon="info">
  Negative spacing utilities toggle the inherent sign of the value definition.
</Hint>

<Preview>
  <div className="d-f fd-c g-2">
    <p className="ls-2 c-slate">Tight spacing (ls-2)</p>
    <p className="ls--2 c-slate">Wide spacing (ls--2)</p>
  </div>
</Preview>

```html "ls-2" "ls--2"
<p class="ls-2 c-slate">Tight spacing (-0.025em)</p>
<p class="ls--2 c-slate">Wide spacing (0.025em)</p>
```
