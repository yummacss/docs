## Usage

### Outset

Use the `bs-o-[value]` utilities to apply outset shadows.

<Preview className="bg-silver-1">
  <div className="d-g g-16 gtc-1 sm:gtc-4">
    <div className="ai-c bg-white bs-o-xs d-f w-18 h-18 jc-c c-slate">xs</div>
    <div className="ai-c bg-white bs-o-sm d-f w-18 h-18 jc-c c-slate">sm</div>
    <div className="ai-c bg-white bs-o-md d-f w-18 h-18 jc-c c-slate">md</div>
    <div className="ai-c bg-white bs-o-lg d-f w-18 h-18 jc-c c-slate">lg</div>
  </div>
</Preview>

```html "bs-o-xs" "bs-o-sm" "bs-o-md" "bs-o-lg"
<div class="d-g g-16 gtc-1 sm:gtc-4">
  <div class="ai-c bg-white bs-o-xs d-f w-18 h-18 jc-c c-slate">xs</div>
```
