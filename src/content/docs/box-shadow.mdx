## Usage

### Outset

Use the `bs-o-*` utilities to apply outset shadows.

<Preview className="bg-silver-1">
  <div className="d-g g-16 gtc-1 sm:gtc-4">
    <div className="d-f ai-c jc-c w-18 h-18 bg-white c-slate bs-o-xs">xs</div>
    <div className="d-f ai-c jc-c w-18 h-18 bg-white c-slate bs-o-sm">sm</div>
    <div className="d-f ai-c jc-c w-18 h-18 bg-white c-slate bs-o-md">md</div>
    <div className="d-f ai-c jc-c w-18 h-18 bg-white c-slate bs-o-lg">lg</div>
  </div>
</Preview>

```html "bs-o-xs" "bs-o-sm" "bs-o-md" "bs-o-lg"
<div class="d-g g-16 gtc-1 sm:gtc-4">
  <div class="d-f ai-c jc-c w-18 h-18 bg-white c-slate bs-o-xs">xs</div>

</div>
```
