## Fixed

> Initial value

Allows the field to size itself based on the content, adjusting its size dynamically to fit the content inside.

<Preview>
  <div className="d-f fd-c">
    <label className="mb-2 c-slate fw-600">Write a message:</label>
    <textarea className="bw-1 bc-silver-4 f:oc-silver-1 f:os-s f:ow-2 fs-f max-h-50 max-w-100 r-none" placeholder="Hello, my name is..."></textarea>
  </div>
</Preview>
