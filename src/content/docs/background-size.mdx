## Usage

### Auto

The background image retains its original size.

<Preview id="background-size-auto" />

```tsx registry="background-size-auto"
```

The background image covers the entire element.

<Preview id="background-size-cover" />

```tsx registry="background-size-cover"
```

The background image scales to fit within the element.

<Preview id="background-size-contain" />

```tsx registry="background-size-contain"
```
