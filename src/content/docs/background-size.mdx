## Usage

### Auto

The background image retains its original size.

<Preview>
  <div className="w-30 h-30">
    <div className="h-30 bs-auto bp-c" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-auto"
<div class="w-30 h-30">
  <div class="h-30 bs-auto bp-c" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```

The background image covers the entire element.

<Preview>
  <div className="w-30 h-30">
    <div className="h-30 bs-c bp-c" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-c"
<div class="w-30 h-30">
  <div class="h-30 bs-c bp-c" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```

The background image scales to fit within the element.

<Preview>
  <div className="w-30 h-30">
    <div className="h-30 bs-co bp-c" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-co"
<div class="w-30 h-30">
  <div class="h-30 bs-co bp-c" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```
