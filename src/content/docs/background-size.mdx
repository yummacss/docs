## Auto

> Initial value

The background image retains its original size.

<Preview>
  <div className="d-30">
    <div className="bs-auto bp-c h-30" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-auto"
<div class="d-30">
  <div class="bs-auto bp-c h-30" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```

## Cover

The background image will cover the entire element.

<Preview>
  <div className="d-30">
    <div className="bs-c bp-c h-30" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-c"
<div class="d-30">
  <div class="bs-c bp-c h-30" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```

## Contain

The background image will be scaled to fit within the element.

<Preview>
  <div className="d-30">
    <div className="bs-co bp-c h-30" style={{ backgroundImage: "url(/img/lighthouse.jpg)" }}></div>
  </div>
</Preview>

```html "bs-co"
<div class="d-30">
  <div class="bs-co bp-c h-30" style="background-image:url(/img/lighthouse.jpg)"></div>
</div>
```
