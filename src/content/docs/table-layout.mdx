## Auto

> Initial value

The browser will determine the column widths based on the content.

<Preview>
  <table className="b-1 b-d bc-c tc-slate tl-auto w-full">
    <thead>
      <tr>
        <th className="b-1 b-d bg-silver-2 bc-slate p-3 ta-c">Full Name</th>
        <th className="b-1 b-d bg-silver-2 bc-slate p-3 ta-c">Phone Number</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td className="b-1 b-d bc-slate p-3 ta-c">John Doe</td>
        <td className="b-1 b-d bc-slate p-3 ta-c">+1 (555) 123-4567</td>
      </tr>
    </tbody>
  </table>
</Preview>
