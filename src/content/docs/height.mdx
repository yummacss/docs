## Max Height

Controls the maximum height of an element.

<Baseline path="css.properties.max-height" />

<Reference category="boxModel" name="max-height" />

The **max-h-64** utility sets the maximum height to `16rem`, ensuring that the element does not exceed this height regardless of the content inside.

<Note icon="arrows_vertical">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Note>

<Preview>
    <div className="ai-c bg-indigo d-f jc-c max-h-64 min-h-16 o-auto p-3 r-v c-white h-32">64</div>
</Preview>

```html "max-h-64"
<div class="ai-c bg-indigo d-f jc-c max-h-64 min-h-16 o-auto p-3 r-v c-white h-32">64</div>
```

## Min Height

Controls the minimum height of an element.

<Baseline path="css.properties.min-height" />

<Reference category="boxModel" name="min-height" />

The **min-h-64** utility sets the minimum height to `8rem`, ensuring that the element maintains at least this height even when the content is smaller.

<Note icon="arrows_vertical">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Note>

<Preview>
    <div className="ai-c bg-indigo d-f jc-c max-h-64 min-h-16 o-auto p-3 r-v c-white h-32">32</div>
</Preview>

```html "max-h-64"
<div class="ai-c bg-indigo d-f jc-c max-h-64 min-h-16 o-auto p-3 r-v c-white h-32">32</div>
```
