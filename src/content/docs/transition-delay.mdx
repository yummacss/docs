## Usage

Use the `tde-*` utilities to specify a delay before a transition effect begins.

<Preview>
  <div className="bg-indigo d-16 tp-a td-300 tde-500 h:bg-indigo-3"></div>
</Preview>

```html "tde-500"
<div class="bg-indigo d-16 tp-a td-300 tde-500 h:bg-indigo-3"></div>
```
