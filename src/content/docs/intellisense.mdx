Speed up development with the official extensions. [Yumma CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=yumma-css.yummacss-intellisense) provides auto-completion, hover previews & smart diagnostics directly in your editor.

## Visual Studio Code

Install the official [VS Code extension](https://marketplace.visualstudio.com/items?itemName=yumma-css.yummacss-intellisense) to enable these features.

## Features

### IntelliSense
Get auto-completion suggestions for abbreviated utility classes as you type.

![IntelliSense auto-completion](/blog/yummacss-intellisense-completion.png)

### Hover Previews
View exact CSS declarations by hovering over any class in your code.

![Hover previews](/blog/yummacss-intellisense-hover.png)

### Smart Diagnostics
Identify overlapping styles & conflicting classes in real-time.

![Smart diagnostics](/blog/yummacss-intellisense-diagnostics.png)

### Quick Fixes
Resolve conflicts with 1-click quick fixes.

![Quick fixes](/blog/yummacss-intellisense-quick-fixes.png)
