## Usage

### Visible

Display the element normally in the layout.

<Preview>
  <div className="d-g g-4 gtc-1 sm:gtc-3">
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white v-v">A</div>
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white">B</div>
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white">C</div>
  </div>
</Preview>

```html "v-v"
<div class="d-g g-4 gtc-1 sm:gtc-3">
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white v-v">A</div>
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white">B</div>
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white">C</div>
</div>
```

### Collapse

Hide the element without taking up space.

  <table className="bw-1 bs-d bc-c c-slate w-full">

      </tr>
    </thead>
    <tbody>
      <tr>
        <td className="bw-1 bs-d bc-slate p-3 ta-c">A</td>

      <tr className="v-c">

    </tbody>
  </table>
</Preview>

### Hidden

Hide the element while still taking up space.

<Preview>
  <div className="d-g g-4 gtc-1 sm:gtc-3">
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white v-h">A</div>
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white">B</div>
    <div className="ai-c bg-indigo d-f jc-c d-16 c-white">C</div>
  </div>
</Preview>

```html "v-h"
<div class="d-g g-4 gtc-1 sm:gtc-3">
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white v-h">A</div>
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white">B</div>
  <div class="ai-c bg-indigo d-f jc-c d-16 c-white">C</div>
</div>
```
