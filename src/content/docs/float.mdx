## Usage

### None

Prevent the element from floating.

<Preview variant="inline">
  <div className="p-r h-14 c-white" id="area">
    <div className="d-f fl-none ai-c jc-c w-14 h-14 bg-indigo">A</div>
  </div>
</Preview>

```html "fl-none"
<div class="p-r h-14 c-white">
  <div class="d-f fl-none ai-c jc-c w-14 h-14 bg-indigo">A</div>
</div>
```
