## Usage

### None

Prevent the element from floating.

<Preview variant="inline">
  <div className="h-14 p-r c-white" id="area">
    <div className="ai-c bg-indigo d-f w-14 h-14 fl-none jc-c">A</div>
  </div>
</Preview>

```html "fl-none"
<div class="h-14 p-r c-white">
  <div class="ai-c bg-indigo d-f w-14 h-14 fl-none jc-c">A</div>
</div>
```
