## None

> Initial value

The element will not float, and it will be displayed in the normal flow of the document.

<Preview variant="inline">
  <div className="h-14 p-r c-white" id="area">
    <div className="ai-c bg-indigo d-f d-14 fl-none jc-c">A</div>
  </div>
</Preview>

```html "fl-none"
<div class="h-14 p-r c-white">
  <div class="ai-c bg-indigo d-f d-14 fl-none jc-c">A</div>
</div>
```

## Inline End

The element will float to the end of the inline direction, which is typically the right side in left-to-right languages.

<Preview variant="inline">
  <div className="h-14 p-r c-white" id="area">
    <div className="ai-c bg-indigo d-f d-14 fl-ie jc-c">A</div>
  </div>
</Preview>

```html "fl-ie"
<div class="h-14 p-r c-white">
  <div class="ai-c bg-indigo d-f d-14 fl-ie jc-c">A</div>
</div>
```

## Inline Start

The element will float to the start of the inline direction, which is typically the left side in left-to-right languages.

<Preview variant="inline">
  <div className="h-14 p-r c-white" id="area">
    <div className="ai-c bg-indigo d-f d-14 fl-is jc-c">A</div>
  </div>
</Preview>

```html "fl-is"
<div class="h-14 p-r c-white">
  <div class="ai-c bg-indigo d-f d-14 fl-is jc-c">A</div>
</div>
```

## Left

The element will float to the left side of its container, allowing text and inline elements to wrap around it.

<Preview variant="inline">
  <div className="h-14 p-r c-white" id="area">
    <div className="ai-c bg-indigo d-f d-14 fl-l jc-c">A</div>
  </div>
</Preview>

```html "fl-l"
<div class="h-14 p-r c-white">
  <div class="ai-c bg-indigo d-f d-14 fl-l jc-c">A</div>
</div>
```

## Right

The element will float to the right side of its container, allowing text and inline elements to wrap around it.
