## Usage

### Auto

Use the `obx-a` utility to allow scroll overflow behavior on the horizontal axis.

```html "obx-a"
<div class="obx-a ...">
  ...
</div>
```

### Contain

Use the `obx-c` utility to contain scroll overflow behavior on the horizontal axis.

```html "obx-c"
<div class="obx-c ...">
  ...
</div>
```

### None

Use the `obx-n` utility to disable scroll overflow behavior on the horizontal axis.

```html "obx-n"
<div class="obx-n ...">
  ...
</div>
```
