## Scroll

> Initial value

The image will move with the content of the page.

<Note icon="cursor_arrow">Try scrolling through the image to see how it behaves.</Note>

<Preview>
    <div className="o-auto ba-s bs-c ff-c fs-xxl fw-700 max-h-80 p-5 ta-c" style={{backgroundImage: 'url(/img/sea.jpg)'}}>{bleakHouse}</div>
</Preview>

