## Usage

### Scroll

The background image moves with the page content.

<Hint icon="mouse_scroll">Scroll through the image to see its behavior.</Hint>

<Preview id="background-attachment-scroll" />

```tsx registry="background-attachment-scroll"
```

### Fixed

The background image remains fixed when scrolling.

<Hint icon="mouse_scroll">Scroll the page to see its behavior.</Hint>

<Preview id="background-attachment-fixed" />

```tsx registry="background-attachment-fixed"
```

### Local

The background image scrolls with the element content.

<Hint icon="mouse_scroll">Scroll through the image to see its behavior.</Hint>

<Preview id="background-attachment-local" />

```tsx registry="background-attachment-local"
```
