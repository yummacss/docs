## Usage

### Auto

Use the `obi-a` utility to allow scroll overflow behavior on the inline axis.

```html "obi-a"
<div class="obi-a ...">
  ...
</div>
```

### Contain

Use the `obi-c` utility to contain scroll overflow behavior on the inline axis.

```html "obi-c"
<div class="obi-c ...">
  ...
</div>
```

### None

Use the `obi-n` utility to disable scroll overflow behavior on the inline axis.

```html "obi-n"
<div class="obi-n ...">
  ...
</div>
```
