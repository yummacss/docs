## Usage

### Auto

Allow the element to receive pointer events as normal.

<Hint icon="cursor_click">Try clicking the search emoji to see how the input reacts.</Hint>

<Preview className="bg-silver-1">
  <div>
    <label className="d-b fw-600 c-slate mb-1">Auto</label>
    <div className="p-r">
      <input className="bw-1 bc-silver-4 f:oc-silver-1 f:os-s f:ow-2 pl-10 pr-4 py-2 w-full" placeholder="Search..." />
      <div className="ai-c d-f b-0 l-0 t-0 p-a pe-auto pl-3">🔎</div>
    </div>
  </div>
</Preview>

```html "pe-auto"
<div>
  <label class="d-b fw-600 c-slate mb-1">Auto</label>
  <div class="p-r">
    <input class="bw-1 bc-gray-2 pl-10 pr-4 py-2 w-full" placeholder="Search..." />
    <div class="ai-c d-f b-0 l-0 t-0 p-a pe-auto pl-3">🔎</div>
  </div>
</div>
```
