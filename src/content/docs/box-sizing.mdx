## Usage

### Content Box

Width & height exclude padding & borders.

<Preview>
  <div className="d-26 bs-cb p-8" id="area">
    <div className="ai-c bg-indigo d-f d-full jc-c c-white">Content</div>
  </div>
</Preview>

```html "bs-cb"
<div class="d-26 bs-cb p-8">
  <div class="ai-c bg-indigo d-f d-full jc-c c-white">Content</div>
</div>
```

### Border Box

Width & height include padding & borders.

<Hint icon="heart">Yumma CSS uses the `border-box` CSS property by default. Learn more about it in the [base styles docs](/docs/base-styles).</Hint>

<Preview>
  <div className="d-26 bs-bb p-8" id="area">
    <div className="ai-c bg-indigo d-f d-full jc-c c-white">Box</div>
  </div>
</Preview>

```html "bs-bb"
<div class="d-26 bs-bb p-8">
  <div class="ai-c bg-indigo d-f d-full jc-c c-white">Box</div>
</div>
```
