## Content Box

> Initial value

The element's width and height do not include padding and borders.

## Border Box

The element's width and height include padding and borders.

<Hint icon="heart">Yumma CSS uses the `border-box` CSS property by default. Learn more about it in the [base styles docs](/docs/base-styles).</Hint>
