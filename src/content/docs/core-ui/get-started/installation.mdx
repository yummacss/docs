## Installing Yumma UI

<Steps>
    1. **Install Yumma UI**

        <PackageManagers pkgManagers={["pnpm", "npm", "yarn"]} icons={false} pkg="@yummacss/ui -D" title="Terminal" />

        <br />

    2. **Import styles**

        Import the Yumma UI styles in your project.

        ```js title="main.ts" mark={1}
        import "@yummacss/ui/styles.css";
        import "./globals.css";
        ```

</Steps>

### Why Yumma UI?

Yumma UI uses [Yumma CSS](https://yummacss.com) as its styling language and [Base UI](https://baseui.com/) under the hood. It gives you a set of reusable components that you can use to build your UIs super quickly. It's lightweight, accessible, and super customizable.

## Next steps
