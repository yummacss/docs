## Installing Yumma UI

<Steps>
    1. **Install Yumma UI**

        <PackageManagers pkgManagers={["pnpm", "npm", "yarn"]} icons={false} pkg="@yummacss/ui -D" title="Terminal" />

        <br />

    2. **Import styles**

        Import the Yumma UI styles in your project.

        ```js title="main.ts" mark={1}
        import "@yummacss/ui/styles.css";
        import "./globals.css";
        ```

</Steps>

### Why Yumma UI?

Yumma UI is a set of accessible and mature React components built on top of Yumma CSS and [Base UI](https://baseui.com/) that you can use to build your UIs super quickly. It's super minimalist, lightweight, and super customizable.

## Next steps

After installing Yumma UI, you can start browsing the [components](/docs/ui/components) section to learn more about the available components.
