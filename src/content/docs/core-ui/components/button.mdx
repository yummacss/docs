## API reference

### `variant`

> Default value is `base`

```ts
(property) variant?: "link" | "destructive" | "base" | "ghost" | "outlined" | null | undefined
```

### `size`

> Default value is `md`

```ts
(property) size?: "sm" | "md" | "lg" | null | undefined
```
