## Example

A component that displays a small piece of information or status indicator.

```tsx showLineNumbers
import { Badge } from "@yummacss/ui";
```

```tsx showLineNumbers startLineNumber=3
export default function Page() {
  return (
    <Badge variant="base" size="xs">New</Badge>
  );
};
```

## API reference

### `variant`

> Default value is `base`

```ts
(property) variant?: "destructive" | "base" | "outline" | null | undefined
```

### `size`

> Default value is `xs`

```ts
(property) size?: "xs" | "sm" | "md" | "lg" | null | undefined
```

## Custom styling

Apply custom styles to the `<Badge>` component.

```tsx showLineNumbers 'className="..."'

  ...
</Badge>
```
