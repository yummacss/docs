## Usage

### Row

Position items horizontally in a single row.

<Preview variant="inline">
  <div className="d-f fd-r cg-4 c-white">
    <div className="d-f ai-c jc-c w-16 h-16 bg-indigo ta-c">A</div>
    <div className="d-f ai-c jc-c w-16 h-16 bg-indigo ta-c">B</div>
    <div className="d-f ai-c jc-c w-16 h-16 bg-indigo ta-c">C</div>
  </div>
</Preview>

