## Why Yumma UI?

Yumma UI is a significant step forward for modern React apps. It's a set of accessible and mature React components built on top of Yumma CSS. With its foundation on top of [Base UI](https://base-ui.com).

:::note[Before you install...]
Please be kind. This is the first version, so there might be a few rough spots. We'll be making improvements as we go. Feel free to [open issues](https://github.com/yumma-lib/yumma-ui/issues) and [pull requests](https://github.com/yumma-lib/yumma-ui/pulls).
:::

### Typescript support

If you use Typescript, you can import the types from `@yummacss/ui`. You can also trigger intellisense for the props of each component in any code editor that supports it.

```ts title="index.tsx"
import type { ButtonProps } from "@yummacss/ui";
```

### Tree shaking

All components in `@yummacss/ui` are tree-shakeable. It's kinda like the same principle as Yumma CSS, meaning you only ship the code you actually use.

```tsx title="index.tsx"
import { Button } from "@yummacss/ui";
import { Button, Card, Input } from "@yummacss/ui";
```

## Getting started

Yumma UI is available as a package on [npm](https://www.npmjs.com/package/@yummacss/ui). You can install it using your favorite package manager:

<Steps>
    1. **Install Yumma UI**

        <PackageManagers pkgManagers={["pnpm", "npm", "yarn"]} icons={false} pkg="@yummacss/ui -D" title="Terminal" />

        <br />

    2. **Import styles**

        Import the Yumma UI styles in your project.

        ```ts title="main.ts" mark={1}
        import "@yummacss/ui/styles.css";
        import "./globals.css";
        ```

    3. **Import components**

        That's pretty much it.

        ```tsx title="index.tsx" mark={1,4}
        import { Button } from "@yummacss/ui";

        export default function App() {
          return <Button variant="base">Login</Button>;
        }
        ```

</Steps>

## What's next?

This 0.1.0 release is just the beginning. It's very likely that we'll add 10 more components for each minor release, but we're also open to community feedback. So, if you have any ideas, please [open an issue](https://github.com/yumma-lib/yumma-ui/issues) or [send us a tweet](https://x.com/yummacss).

- **More components**: Combobox, Toast, Tooltip, Menubar, and much more
- **Theming component**: Customize colors, typography, radii, and other design choices
- **Framework support**: Support for other frameworks like Astro
- And much more...

## Closing thoughts

We can't wait to see what you build with Yumma UI! Remember, this is our first release and is currently only meant for React frameworks, but we're working on support for others.
