**This update shouldn't be difficult to upgrade, but you may want to check out the [release notes](https://github.com/yumma-lib/yumma-css/releases/tag/v3.5.0) before upgrading.**

---

## First-party engine

That's right, we decided to create our own first-party engine for Yumma CSS. This shouldn't have any impact on your projects, but it's a big deal for us internally.

This change will allow us to extend Yumma CSS to feature projects like [Yumma CSS Play](https://play.yummacss.com) and more. Now of course this is a very specific change, but it's a big one for us internally.

---

## Opacity variants

Now you can use opacity variants for all color utilities. This is super useful for targeting opacity to different utilities.

```html live "bg-indigo/5" "bg-indigo/10" "bg-indigo/25" "bg-indigo/50"
<div class="d-g g-1 gtc-4">
  <div class="bg-indigo/5 d-16 rad-1"></div>
  <div class="bg-indigo/10 d-16 rad-1"></div>
  <div class="bg-indigo/25 d-16 rad-1"></div>
  <div class="bg-indigo/50 d-16 rad-1"></div>
</div>
```
