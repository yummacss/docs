## Usage

### Normal

Use the `wb-n` utility to use default line break rules.

```html "wb-n"
<p class="wb-n ...">
  ...
</p>
```

### Break All

Use the `wb-ba` utility to allow word breaks between any two characters.

```html "wb-ba"
<p class="wb-ba ...">
  ...
</p>
```

### Keep All

Use the `wb-ka` utility to prevent word breaks for CJK texts.

```html "wb-ka"
<p class="wb-ka ...">
  ...
</p>
```
