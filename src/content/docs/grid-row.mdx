## Grid Row Span

Controls the row span of a grid item.

<Baseline path="css.properties.grid-row" />

<UtilityTable category="grid" name="grid-row" />

The item will span across **2** rows, starting from its current position and occupying the next **2** rows in the grid.

## Grid Row End

Controls where the grid item ends on the grid rows.

<Baseline path="css.properties.grid-row-end" />

<UtilityTable category="grid" name="grid-row-end" />

The item will end at the first row line, effectively spanning from its starting position to the second row line.

<Preview variant="inline">
    <div className="p-r">
      <div className="d-g g-4 gaf-c gtr-3 ta-c" id="area">
          <div className="ai-c bg-indigo d-f d-g gr-s-2 gre-3 jc-c p-4 c-white">A</div>
          <div className="ai-c bg-indigo-8 d-f d-g gre-4 grs-1 jc-c p-4 c-indigo-5">B</div>
          <div className="ai-c bg-indigo-8 d-f d-g gre-4 grs-1 jc-c p-4 c-indigo-5">C</div>
      </div>
    </div>
</Preview>

```html "gre-3"
<div class="p-r">
  <div class="d-g g-4 gaf-c gtr-3 ta-c">
    <div class="ai-c bg-indigo d-f d-g gr-s-2 gre-3 jc-c p-4 c-white">A</div>
    <div class="ai-c bg-indigo-8 d-f d-g gre-4 grs-1 jc-c p-4 c-indigo-5">B</div>
```

## Grid Row Start

Controls where the grid item begins on the grid rows.

<Baseline path="css.properties.grid-row-start" />

<UtilityTable category="grid" name="grid-row-start" />

The item will begin at the first row line, positioning it to start in the third row of the grid.

<Preview variant="inline">
    <div className="p-r">

          <div className="ai-c bg-indigo-8 d-f d-g gre-4 grs-1 jc-c p-4 c-indigo-5">B</div>
          <div className="ai-c bg-indigo-8 d-f d-g gre-4 grs-1 jc-c p-4 c-indigo-5">C</div>
      </div>
    </div>
</Preview>
