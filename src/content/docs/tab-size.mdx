## Tab Size 2

The tab size will be set to 2, which is a common indentation size for code.

<Preview variant="inline">
    <section className="bg-black">
        <pre className="fs-md ts-2">This text is indented with a tab character.</pre>
    </section>
</Preview>

```html "ts-2"
<section class="bg-black">
    <pre class="fs-md ts-2">This text is indented with a tab character.</pre>
</section>
```
