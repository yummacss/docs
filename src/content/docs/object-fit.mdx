## Usage

### Fill

Stretch the content to fill the entire container.

<Preview>
  <div className="h-40 w-80">
    <img className="of-f w-full h-full" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-f"
<div class="h-40 w-80">
  <img class="of-f w-full h-full" src="/img/coast.jpg" />
</div>
```

### Cover

Scale the content to cover the entire container.

<Preview>
  <div className="h-40 w-80">
    <img className="of-c w-full h-full" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-c"
<div class="h-40 w-80">
  <img class="of-c w-full h-full" src="/img/coast.jpg" />
</div>
```

### None

Display the content at its original size.

<Preview>
  <div className="h-40 w-80">
    <img className="of-none w-full h-full" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-none"
<div class="h-40 w-80">
  <img class="of-none w-full h-full" src="/img/coast.jpg" />
</div>
```

### Scale Down

Size the content as if `none` or `contain` was specified.

<Preview>
  <div className="h-40 w-80">
    <img className="of-sd w-full h-full" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-sd"
<div class="h-40 w-80">
  <img class="of-sd w-full h-full" src="/img/coast.jpg" />
</div>
```
