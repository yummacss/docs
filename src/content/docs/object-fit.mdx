## Fill

> Initial value

The content will stretch to fill the entire container, disregarding its aspect ratio.

<Preview>
  <div className="h-40 w-80" id="area">
    <img className="d-full of-f" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-f"
<div class="h-40 w-80">
  <img class="d-full of-f" src="/img/coast.jpg" />
</div>
```

## Cover

The content will scale to cover the entire container while maintaining its aspect ratio, potentially cropping the content.

<Preview>
  <div className="h-40 w-80" id="area">
    <img className="d-full of-c" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-c"
<div class="h-40 w-80">
  <img class="d-full of-c" src="/img/coast.jpg" />
</div>
```

## None

The content will not be resized to fit the container, and it will retain its original size.

<Preview>
  <div className="h-40 w-80" id="area">
    <img className="d-full of-none" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-none"
<div class="h-40 w-80">
  <img class="d-full of-none" src="/img/coast.jpg" />
</div>
```

## Scale Down

The content will be sized as if `none` or `contain`, whichever results in a smaller size.

<Preview>
  <div className="h-40 w-80" id="area">
    <img className="d-full of-sd" src="/img/coast.jpg" />
  </div>
</Preview>

```html "of-sd"
<div class="h-40 w-80">
  <img class="d-full of-sd" src="/img/coast.jpg" />
</div>
```
