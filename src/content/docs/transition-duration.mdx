## Usage

Use the `tdu-*` utilities to control the amount of time a transition should take to complete.

<Preview>
  <div class="d-16 bg-indigo tp-a td-3 tdu-3 ttf-io h:t-s-80 h:bg-slate-7"></div>
</Preview>

