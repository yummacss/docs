## Use color utilities

Apply color system utilities to style your elements.

  <div className="d-g g-16 gtc-1 sm:gtc-3">
    <div className="bg-red d-16 br-1"></div>
    <div className="bg-green d-16 br-1"></div>
    <div className="bg-blue d-16 br-1"></div>
  </div>
</Preview>

```html "bg-red" "bg-green" "bg-blue"
<div class="d-g g-16 gtc-1 sm:gtc-3">
  <div class="bg-red d-16"></div>
  <div class="bg-green d-16"></div>
  <div class="bg-blue d-16"></div>
</div>
```

## Use opacity modifiers

Target specific opacity values across color utilities. Opacity variants range from 0 to 95 in increments of 5.

<Preview>
  <div className="d-g g-16 gtc-1 sm:gtc-3">

    <div className="bg-indigo/60 d-16 br-1"></div>
    <div className="bg-indigo-9/40 d-16 br-1"></div>
  </div>
</Preview>

```html "bg-indigo-3/50" "bg-indigo/25" "bg-indigo-9/20"

  <div class="bg-indigo/60 d-16"></div>
```
