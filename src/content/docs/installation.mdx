## Setting up Yumma CSS

Here's a very quick and easy way to get started with Yumma CSS.

### With package managers

Yumma CSS is available as a package on [npm](https://www.npmjs.com/package/yummacss). You can install it using your favorite package manager:

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash title="Terminal"
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash title="Terminal"
    pnpm yummacss init
    ```
  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/styles.css",
      buildOptions: {
        reset: true,
        minify: false,
      },
    };
    ```
  </Step>

  <Step title="Build styles">
    You can now start generating your CSS with the [`build`](/docs/foundations/config/#build) or [`watch`](/docs/foundations/config/#watch) command.

    ```bash title="Terminal"
    pnpm yummacss build
    ```
  </Step>
</Stepper>

### With CDN

Yumma CSS is also available as a CDN. You can import it directly in your HTML file.

```html title="index.html" {7, 10-12}
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://cdn.jsdelivr.net/npm/@yummacss/runtime@0.1.0/dist/index.iife.js"></script>
</head>

<body class="bg-black">
    <h1 class="tc-white">Hello World</h1>
</body>

</html>
```
