## Set up Yumma CSS

Here's a very quick and easy way to get started with Yumma CSS.

### Use the CLI

Yumma CSS is available as a package on [npm](https://www.npmjs.com/package/yummacss). You can install it using your favorite package manager:

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash title="Terminal"
    npm install yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash title="Terminal"
    npx yummacss init
    ```

  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/styles.css",
      buildOptions: {
        reset: true,
        minify: false,
      },
    };
    ```

  </Step>

  <Step title="Build styles">
    You're all set! Start typing Yumma CSS classes, and watch the magic happening in our CSS file.

    ```bash title="Terminal"
    npx yummacss build
    ```

  </Step>
</Stepper>

### Use the CDN

Yumma CSS is also available as a CDN. You can import it directly in your HTML file.

<Stepper>
  <Step title="Import Yumma CSS">

    ```html
    <script src="https://unpkg.com/@yummacss/runtime"></script>
    ```

  </Step>

  <Step title="Start using Yumma CSS">

    ```html title="index.html" {7, 10-12}
    <!DOCTYPE html>
    <html lang="en">

    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <script src="https://unpkg.com/@yummacss/runtime"></script>
    </head>

    <body class="bg-black">
      <h1 class="c-white">Hello World</h1>
    </body>

    </html>
    ```

  </Step>
</Stepper>
