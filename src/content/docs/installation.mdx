## Set up Yumma CSS

Follow these steps to get started with Yumma CSS.

### Use the CLI

Install Yumma CSS with your favorite package manager:

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash title="Terminal"
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash title="Terminal"
    pnpx yummacss init
    ```

  </Step>

  <Step title="Configure sources">
    Specify the locations/paths of your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/styles.css",
      buildOptions: {
        reset: true,
        minify: false,
      },
    };
    ```

  </Step>

  <Step title="Build styles">
    Start using Yumma CSS classes, then generate your styles.

    ```bash title="Terminal"
    pnpx yummacss build
    ```

  </Step>
</Stepper>

### Use the CDN

Import Yumma CSS directly into your HTML file via CDN for quick prototyping.

<Stepper>
  <Step title="Import script">

    ```html
    <script src="https://unpkg.com/@yummacss/runtime"></script>
    ```

  </Step>

  <Step title="Start styling">

    ```html title="index.html" {7, 10-12}
    <!DOCTYPE html>
    <html lang="en">

    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <script src="https://unpkg.com/@yummacss/runtime"></script>
    </head>

    <body class="bg-black">
      <h1 class="c-white">Hello World</h1>
    </body>

    </html>
    ```

  </Step>
</Stepper>
