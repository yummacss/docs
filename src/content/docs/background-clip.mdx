## Usage

### Border Box

The background image extends to the outer edge of the border.

<Preview>
  <div className="w-20 h-20 p-2 bc-bb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
</Preview>

```html "bc-bb"
<div class="w-20 h-20 p-2 bc-bb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
```

### Content Box

The background image clips to the content area, excluding padding & borders.

<Preview>
  <div className="w-20 h-20 p-2 bc-cb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
</Preview>

```html "bc-cb"
<div class="w-20 h-20 p-2 bc-cb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
```

### Padding Box

The background image extends to the outer edge of the padding, excluding the border.

<Preview>
  <div className="w-20 h-20 p-2 bc-pb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
</Preview>

```html "bc-pb"
<div class="w-20 h-20 p-2 bc-pb bg-indigo bc-indigo-8 bw-3 bs-d"></div>
```

### Text

The background image clips to the text.

<Preview>
  <div className="o-h px-4 py-2 bc-t c-transparent br-2 fs-3xl fw-700" style={{ backgroundImage: "linear-gradient(132deg, #413cb8 0.00%, #7a77cd 100.00%)" }}>
    Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall.
  </div>
</Preview>

```html "bc-t"
<div class="o-h px-4 py-2 bc-t c-transparent br-2 fs-3xl fw-700" style="background-image: linear-gradient(132deg, #413cb8 0.00%, #7a77cd 100.00%);">
  London. Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall.
</div>
```
