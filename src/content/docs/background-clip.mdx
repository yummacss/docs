## Border Box

> Initial value

The background extends to the outer edge of the border (including the border itself).

<Preview>
  <div className="b-3 bs-d bc-bb bc-indigo-8 bg-indigo d-20 p-2"></div>
</Preview>

```html "bc-bb"
<div class="b-3 bs-d bc-bb bc-indigo-8 bg-indigo d-20 p-2"></div>
```

## Content Box

The background is confined to the content area, excluding padding and borders.

<Preview>
  <div className="b-3 bs-d bc-cb bc-indigo-8 bg-indigo d-20 p-2"></div>
</Preview>

```html "bc-cb"
<div class="b-3 bs-d bc-cb bc-indigo-8 bg-indigo d-20 p-2"></div>
```

## Padding Box

The background extends to the outer edge of the padding, but does not include the border.

<Preview>
  <div className="b-3 bs-d bc-indigo-8 bc-pb bg-indigo d-20 p-2"></div>
</Preview>

```html "bc-pb"
<div class="b-3 bs-d bc-indigo-8 bc-pb bg-indigo d-20 p-2"></div>
```

## Text

The background is clipped to the text itself, allowing for unique visual effects.

<Preview>
  <div className="bc-t fs-3xl fw-700 o-h px-4 py-2 rad-2 c-transparent" style={{ backgroundImage: "linear-gradient(132deg, #413cb8 0.00%, #7a77cd 100.00%)" }}>
    Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall.
  </div>
</Preview>

```html "bc-t"
<div class="bc-t fs-3xl fw-700 o-h px-4 py-2 rad-2 c-transparent" style="background-image: linear-gradient(132deg, #413cb8 0.00%, #7a77cd 100.00%);">
  London. Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall.
</div>
```
