## Usage

### Border Box

The background image extends to the outer edge of the border.

<Preview id="background-clip-border" />

```tsx registry="background-clip-border"
```

### Content Box

The background image clips to the content area, excluding padding & borders.

<Preview id="background-clip-content" />

```tsx registry="background-clip-content"
```

### Padding Box

The background image extends to the outer edge of the padding, excluding the border.

<Preview id="background-clip-padding" />

```tsx registry="background-clip-padding"
```

### Text

The background image clips to the text.

<Preview id="background-clip-text" />

```tsx registry="background-clip-text"
```
