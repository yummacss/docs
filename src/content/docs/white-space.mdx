## Normal

> Initial value

The text will wrap normally, and multiple spaces will be collapsed into a single space.

<Preview>
  <div className="c-slate fs-lg ws-n">{bleakHouse}</div>
</Preview>

```html "ws-n"
<div class="c-slate fs-lg ws-n">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Break Spaces

The text will respect spaces and line breaks, allowing for wrapping at spaces.

<Preview>
  <div className="c-slate fs-lg ws-bs">{bleakHouse}</div>
</Preview>

```html "ws-bs"
<div class="c-slate fs-lg ws-bs">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## No Wrap

The text will not wrap to the next line, and it will overflow the container if it exceeds the width.

<Preview>
  <div className="o-x-auto">
    <div className="c-slate fs-lg ws-nw">{bleakHouse}</div>
  </div>
</Preview>

```html "ws-nw"
<div class="o-x-auto">
  <div class="c-slate fs-lg ws-nw">
    Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
  </div>
</div>
```

## Pre

The text will be displayed as is, preserving spaces and line breaks.

<Preview className="bg-white o-auto">
  <div className="c-slate fs-lg ws-p">{bleakHouse}</div>
</Preview>

```html "ws-p"
<div class="c-slate fs-lg ws-p">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Pre-Line

The text will preserve line breaks but collapse multiple spaces into a single space.

<Preview>
  <div className="c-slate fs-lg ws-pl">{bleakHouse}</div>
</Preview>

```html "ws-pl"
<div class="c-slate fs-lg ws-pl">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Pre-Wrap

The text will preserve spaces and line breaks, and it will wrap when necessary.

<Preview>
  <div className="c-slate fs-lg ws-pw">{bleakHouse}</div>
</Preview>

```html "ws-pw"
<div class="c-slate fs-lg ws-pw">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```
