## Usage

### Auto

Align the item based on the container's default alignment properties.

<Preview variant="inline">
  <div id="area">
    <div className="d-g g-4 gtc-3 pi-st">
      <div className="ai-c bg-indigo-9 d-14 d-f jc-c c-indigo-5">A</div>
      <div className="ai-c bg-indigo d-14 d-f jc-c ps-auto c-white">B</div>
      <div className="ai-c bg-indigo-9 d-14 d-f jc-c c-indigo-5">C</div>
    </div>
  </div>
</Preview>

