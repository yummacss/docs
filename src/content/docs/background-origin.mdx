## Padding Box

> Initial value

The background positioning area extends to the outer edge of the padding, but does not include the border.

```html live "bo-pb"
<div class="b-3 b-d bc-pb bo-pb bc-indigo-4 bg-indigo d-20 p-2 rad-1"></div>
```

## Border Box

The background positioning area includes the border area.

```html live "bo-bb"
<div class="b-3 b-d bc-bb bo-bb bc-indigo-4 bg-indigo d-20 p-2 rad-1"></div>
```

## Content Box

The background positioning area is confined to the content area, excluding padding and borders.

```html live "bo-cb"
<div class="b-3 b-d bc-cb bo-cb bc-indigo-4 bg-indigo d-20 p-2 rad-1"></div>
```
