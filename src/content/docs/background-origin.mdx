## Usage

### Padding Box

The background image positioning area extends to the outer edge of the padding, excluding the border.

<Preview>
  <div className="w-20 h-20 p-2 bc-pb bo-pb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
</Preview>

```html "bo-pb"
<div class="w-20 h-20 p-2 bc-pb bo-pb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
```

The background image positioning area includes the border area.

<Preview>
  <div className="w-20 h-20 p-2 bc-bb bo-bb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
</Preview>

```html "bo-bb"
<div class="w-20 h-20 p-2 bc-bb bo-bb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
```

The background image positioning area clips to the content area, excluding padding & borders.

<Preview>
  <div className="w-20 h-20 p-2 bc-cb bo-cb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
</Preview>

```html "bo-cb"
<div class="w-20 h-20 p-2 bc-cb bo-cb bg-indigo bc-indigo-4 bw-3 bs-d"></div>
```
