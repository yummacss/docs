## Padding Box

> Initial value

The background positioning area extends to the outer edge of the padding, but does not include the border.

<Preview>
  <div className="b-3 bs-d bc-pb bo-pb bc-indigo-4 bg-indigo d-20 p-2"></div>
</Preview>

```html "bo-pb"
<div class="b-3 bs-d bc-pb bo-pb bc-indigo-4 bg-indigo d-20 p-2"></div>
```

## Border Box

The background positioning area includes the border area.

<Preview>
  <div className="b-3 bs-d bc-bb bo-bb bc-indigo-4 bg-indigo d-20 p-2"></div>
</Preview>

```html "bo-bb"
<div class="b-3 bs-d bc-bb bo-bb bc-indigo-4 bg-indigo d-20 p-2"></div>
```

## Content Box

The background positioning area is confined to the content area, excluding padding and borders.

<Preview>
  <div className="b-3 bs-d bc-cb bo-cb bc-indigo-4 bg-indigo d-20 p-2"></div>
</Preview>

```html "bo-cb"
<div class="b-3 bs-d bc-cb bo-cb bc-indigo-4 bg-indigo d-20 p-2"></div>
```
