## Usage

### Padding Box

The background image positioning area extends to the outer edge of the padding, excluding the border.

<Preview>
  <div className="bw-3 bs-d bc-pb bo-pb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
</Preview>

```html "bo-pb"
<div class="bw-3 bs-d bc-pb bo-pb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
```

The background image positioning area includes the border area.

<Preview>
  <div className="bw-3 bs-d bc-bb bo-bb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
</Preview>

```html "bo-bb"
<div class="bw-3 bs-d bc-bb bo-bb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
```

The background image positioning area clips to the content area, excluding padding & borders.

<Preview>
  <div className="bw-3 bs-d bc-cb bo-cb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
</Preview>

```html "bo-cb"
<div class="bw-3 bs-d bc-cb bo-cb bc-indigo-4 bg-indigo w-20 h-20 p-2"></div>
```
