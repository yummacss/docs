## Usage

### Padding Box

The background image positioning area extends to the outer edge of the padding, excluding the border.

<Preview id="background-origin-padding" />

```tsx registry="background-origin-padding"
```

The background image positioning area includes the border area.

<Preview id="background-origin-border" />

```tsx registry="background-origin-border"
```

The background image positioning area clips to the content area, excluding padding & borders.

<Preview id="background-origin-content" />

```tsx registry="background-origin-content"
```
