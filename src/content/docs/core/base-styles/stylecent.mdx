## Base Styles

Stylecent is a set of styles for Yumma CSS projects, meant for consistent design across browsers.

### Universal selector

Ensures consistent box sizing behavior across all elements.

```scss
*,
*::before,
*::after {
  border-style: solid;
  border-width: 0;
  box-sizing: border-box;
}

* {
  color: inherit;
  font-family: $yma-font-system;
  margin: 0;
}
```

### Typography

Ensures that the default text elements are consistent in the `<body>`.

```scss
body {
  font-family: inherit;
  line-height: 1.5;
}
```

### Media elements

Ensures `<img>`, `<video>`, and other media elements are responsive by default.

```scss
canvas,
img,
picture,
svg,
video {
  display: block;
  max-width: 100%;
}
```

### Form elements

Applies consistent font styles to form elements for better integration with the page.

```scss
button,
input,
optgroup,
select,
textarea {
  background-color: transparent;
  font: inherit;
}
```

### Text wrapping

Ensures long text content wraps appropriately within elements.

```scss
h1,
h2,
h3,
h4,
h5,
h6,
p {
  font-size: inherit;
  font-weight: inherit;
  overflow-wrap: break-word;
}
```

### Link decoration

Removes default text decoration from links for cleaner appearance.

```scss
a {
  text-decoration: none;
}
```

### List normalization

Removes default padding and `list-style` from ordered and unordered lists.

```scss
ol,
ul {
  list-style: none;
  padding: 0;
}
```

### Horizontal rule

Applies a subtle border to horizontal rules for visual separation.

```scss
hr {
  border-top: $yma-colors-transparent;
  border: 0;
  margin: 1em 0;
}
```

## Disabling Stylecent

If you want to disable Stylecent completely and use Yumma CSS, you can modify your import as follows:

### Standard

Disabling Stylecent using standard CSS:

```diff lang="css" title="/src/app.css"
- /* Standard Version */
- @import "https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@latest/dist/yumma.css";

+ /* Standard Core Version */
+ @import "https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@latest/dist/yumma-core.css";
```

### Minified

Disabling Stylecent using minified CSS:

```diff lang="css" title="/src/app.css"
- /* Minified Version */
- @import "https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@latest/dist/yumma.min.css";

+ /* Minified Core Version */
+ @import "https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@latest/dist/yumma-core.min.css";
```
