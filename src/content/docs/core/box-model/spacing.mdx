Controls the spacing of child elements.

## Spacing X

Controls the left and right X-axis margins of child elements.

<Utility
  classPrefix="s-x-"
  incrementValue={0.25}
  propNames={["margin-left", "margin-right"]}
  range={100}
  unit="rem"
/>

```html live "s-x-8"
<div class="area d-if fd-r s-x-8">
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">A</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">B</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">C</div>
</div>
```

## Spacing Y

Controls the top and bottom Y-axis margins of child elements.

<Utility
  classPrefix="s-y-"
  incrementValue={0.25}
  propNames={["margin-top", "margin-bottom"]}
  range={100}
  unit="rem"
/>

```html live "s-y-8"
<div class="area d-if fd-c s-y-8">
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">A</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">B</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">C</div>
</div>
```
