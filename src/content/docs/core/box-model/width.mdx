## Using Percentages

Percentage utilities let you change an element's height based on its container size.

```html live "w-full" "w-half"
<div class="b-3 b-d bc-indigo p-2 rad-2 s-y-8 w-64">
  <div class="ai-c bg-indigo d-f fw-700 jc-c p-2 rad-1 tc-white w-full">100%</div>
  <div class="ai-c bg-indigo d-f fw-700 jc-c p-2 rad-1 tc-white w-half">50%</div>
</div>
```
