## Max Dimension

Controls the maximum height and max width of an element at once.

<Utility
  additionalClasses={[
    { name: "1/1", value: "100dvh" },
    { name: "1/2", value: "50dvh" },
    { name: "auto", value: "auto" },
    { name: "fc", value: "fit-content" },
    { name: "full", value: "100%" },
    { name: "half", value: "50%" },
    { name: "max", value: "max-content" },
    { name: "min", value: "min-content" },
  ]}
  classPrefix="max-dim-"
  incrementValue={0.25}
  propNames={["max-height", "max-width"]}
  range={100}
  unit="rem"
/>

This example sets the maximum height and width of an element to **16rem**. The **max-dim-64** utility ensures the element does not exceed a size of 16rem in both height and width.

<Decorator name="scaling">Try changing the vertical size of the box to see how wide it can be at the maximum size.</Decorator>

```html "max-dim-64"
<div class="ovf-auto ai-c bg-indigo dim-16 d-f jc-c max-dim-64 min-dim-32 p-6 r-b rad-1 tc-white">64</div>
```
