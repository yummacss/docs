Controls how single items are justified and aligned simultaneously.

<Class name="/place-self" />

## Auto

Sets the `place-self` property to `auto`.

```html live "ps-aut layout="/src/layouts/wide.astro"
<div class="stripes rad-1 tc-white">
  <div class="d-g g-4 gtc-2 h-50">
    <div class="bg-indigo d-14 d-f ps-auto ai-c jc-c p-4 rad-1">A</div>
    <div class="bg-indigo d-14 d-f ps-auto ai-c jc-c p-4 rad-1">B</div>
    <div class="bg-indigo d-14 d-f ps-auto ai-c jc-c p-4 rad-1">C</div>
    <div class="bg-indigo d-14 d-f ps-auto ai-c jc-c p-4 rad-1">D</div>
  </div>
</div>
```

## Center

Aligns the item to the center of the grid container.

```html live "ps-c" layout="/src/layouts/wide.astro"
<div class="stripes rad-1 tc-white">
  <div class="d-g g-4 gtc-2 h-50">
    <div class="bg-indigo d-14 d-f ps-c ai-c jc-c p-4 rad-1">A</div>
    <div class="bg-indigo d-14 d-f ps-c ai-c jc-c p-4 rad-1">B</div>
    <div class="bg-indigo d-14 d-f ps-c ai-c jc-c p-4 rad-1">C</div>
    <div class="bg-indigo d-14 d-f ps-c ai-c jc-c p-4 rad-1">D</div>
  </div>
</div>
```
