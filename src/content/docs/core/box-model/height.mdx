Controls the height of an element.

<Utility
  additionalClasses={[
    { name: "1/1", value: "100dvh" },
    { name: "1/2", value: "50dvh" },
    { name: "auto", value: "auto" },
    { name: "fc", value: "fit-content" },
    { name: "full", value: "100%" },
    { name: "half", value: "50%" },
    { name: "max", value: "max-content" },
    { name: "min", value: "min-content" },
  ]}
  classPrefix="h-"
  incrementValue={0.25}
  propNames={["height"]}
  range={100}
  unit="rem"
/>

```html live "h-8" "h-16" "h-24" "h-32"
<div class="d-g g-16 gtc-4">
  <div class="ai-c bg-indigo d-f h-8 jc-c p-2 rad-1 tc-white">2rem</div>
  <div class="ai-c bg-indigo d-f h-16 jc-c p-2 rad-1 tc-white">4rem</div>
  <div class="ai-c bg-indigo d-f h-24 jc-c p-2 rad-1 tc-white">6rem</div>
  <div class="ai-c bg-indigo d-f h-32 jc-c p-2 rad-1 tc-white">8rem</div>
</div>
```

## Using Percentages

Control the height percentage of an element.

```html live "h-full" "h-half"
<div class="b-3 b-d bc-indigo d-g g-16 gtc-2 h-64 p-2 rad-1">
  <div class="ai-c bg-indigo d-f h-full jc-c p-2 rad-1 tc-white">100%</div>
  <div class="ai-c bg-indigo d-f h-half jc-c p-2 rad-1 tc-white">50%</div>
</div>
```
