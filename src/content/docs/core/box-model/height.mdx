## Using Percentages

Control the height percentage of an element.

```html live "h-full" "h-half"
<div class="b-3 b-d bc-indigo d-g g-16 gtc-2 h-64 p-2 rad-1 stripes">
  <div class="ai-c bg-indigo d-f h-full jc-c p-2 rad-1 tc-white">100%</div>
  <div class="ai-c bg-indigo d-f h-half jc-c p-2 rad-1 tc-white">50%</div>
</div>
```
