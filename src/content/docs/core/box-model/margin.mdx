Controls the margin of an element.

<Utility
  additionalClasses={[{ name: "auto", value: "auto" }]}
  classPrefix="m-"
  incrementValue={0.25}
  propNames={["margin"]}
  range={100}
  unit="rem"
/>

```html live "m-8"
<div class="d-f fd-r rad-1 stripes">
  <div class="ai-c bg-indigo d-f dim-16 jc-c m-8 rad-1 tc-white">2rem</div>
</div>
```
