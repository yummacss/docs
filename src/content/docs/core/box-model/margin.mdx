<Class name="margin" />

This example sets the margin to **2rem**. The element will have a uniform margin around it.

```html live "m-8"
<div class="d-f fd-r" id="area">
  <div class="ai-c bg-indigo d-f d-16 jc-c m-8 rad-1 tc-white">8</div>
</div>
```
