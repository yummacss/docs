This example sets the margin to **2rem**. The element will have a uniform margin around it.

```html live "m-8"
<div class="d-f fd-r rad-1 stripes">
  <div class="ai-c bg-indigo d-f dim-16 jc-c m-8 rad-1 tc-white">8</div>
</div>
```

## Margin Bottom
