<Class name="padding" />

This example sets the padding to **2rem**. The element will have uniform padding inside it.

```html live "p-6"
<div class="stripes">
  <div class="p-8">
    <p class="bg-indigo tc-white p-6">6</p>
  </div>
</div>
```
