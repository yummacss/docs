Controls the padding of an element.

<Utility
  additionalClasses={[{ name: "auto", value: "auto" }]}
  classPrefix="p-"
  incrementValue={0.25}
  propNames={["padding"]}
  range={100}
  unit="rem"
/>

```html live "p-8"
<div class="fw-700 tc-white">
  <div class="area rad-2">
    <div class="p-8">
      <p class="bg-indigo tc-white p-4">2rem</p>
    </div>
  </div>
</div>
```
