## Bottom

Controls the bottom placement of positioned elements.

<Class name="bottom" />

```html live "bo-0"

  <div class="p-a i-0 stripes rad-2 bo-1"></div>
  <div class="p-a bo-0 l-0 d-f d-16 ai-c jc-c rad-2 bg-indigo p-4">A</div>
</div>
```
