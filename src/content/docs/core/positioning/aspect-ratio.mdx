## Auto

> Initial value

Allows the element to maintain its default aspect ratio based on its content.

```html live "ar-auto"
<div class="ar-auto h-32">
  <img class="d-full of-c rad-1" src="/img/frangipani.jpg" />
</div>
```

## 1/1

Enforces a square aspect ratio.

```html live "ar-1/1"
<div class="ar-1/1 h-32">
  <img class="d-full of-c rad-1" src="/img/frangipani.jpg" />
</div>
```

## 1/2
