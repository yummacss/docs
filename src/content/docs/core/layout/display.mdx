## Block

This example sets the display to **block**. The element will take up the full width available, and a new line will be started after it.

```html live "d-b" layout="/src/layouts/wide.astro"
<div class="d-b s-y-4 stripes tc-white">
  <div class="bg-indigo p-4 rad-1 ta-c">A</div>
  <div class="bg-indigo p-4 rad-1 ta-c">B</div>
  <div class="bg-indigo p-4 rad-1 ta-c">C</div>
</div>
```

## Flex

This example sets the display to **flex**. The element will use the flexbox layout model, allowing for flexible and responsive layouts.

```html live "d-f" layout="/src/layouts/wide.astro"
<div class="d-f g-4 stripes tc-white">
  <div class="ai-c bg-indigo d-b d-g dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-b d-g dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-b d-g dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Flow Root

This example sets the display to **flow-root**. The element will establish a new block formatting context, allowing for better control of floated children.

```html live "d-fr" layout="/src/layouts/wide.astro"
<div class="d-fr tc-white">
  <div class="ai-c bg-indigo d-f dim-18 flo-l jc-c p-a rad-1 zi-10">A</div>
  <div class="h-18 stripes"></div>
</div>
```

## Grid

This example sets the display to **grid**. The element will use the CSS Grid layout model, enabling the creation of complex two-dimensional layouts.

```html live "d-g" layout="/src/layouts/wide.astro"
<div class="d-g g-4 gtc-2 stripes tc-white">
  <div class="bg-indigo p-4 rad-1 ta-c">A</div>
  <div class="bg-indigo p-4 rad-1 ta-c">B</div>
  <div class="bg-indigo p-4 rad-1 ta-c">C</div>
</div>
```

## Inline Block

This example sets the display to **inline-block**. The element will take up only as much width as necessary, but it can have width and height set, and it will not start a new line.

```html live "d-ib" layout="/src/layouts/wide.astro"
<div class="stripes tc-white">
  <span class="bg-indigo d-ib p-4 rad-1 ta-c w-32">A</span>
  <span class="bg-indigo d-ib p-4 rad-1 ta-c w-32">B</span>
  <span class="bg-indigo d-ib p-4 rad-1 ta-c w-32">C</span>
</div>
```

## Inline Flex

This example sets the display to **inline-flex**. The element will use the flexbox layout model while remaining inline, allowing for flexible layouts without breaking the flow.

```html live "d-if" layout="/src/layouts/wide.astro"
<div class="d-if stripes g-4 tc-white">
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">A</div>
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">B</div>
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">C</div>
</div>
```

## Inline Grid

This example sets the display to **inline-grid**. The element will use the CSS Grid layout model while remaining inline, allowing for grid layouts without breaking the flow.

```html live "d-ig"
<div class="d-ig gtc-2 g-4 stripes tc-white">
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">A</div>
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">B</div>
  <div class="bg-indigo d-ib p-4 rad-1 ta-c w-32">C</div>
</div>
```

## None

This example sets the display to **none**. The element will not be displayed at all, and it will not take up any space in the layout.

```html live "d-none" layout="/src/layouts/wide.astro"
<div class="d-b stripes s-y-4 tc-white">
  <div class="bg-indigo d-none p-4 rad-1 ta-c">A</div>
  <div class="bg-indigo p-4 rad-1 ta-c">B</div>
  <div class="bg-indigo p-4 rad-1 ta-c">C</div>
</div>
```
