<Class name="visibility" />

## Collapse

This example sets the visibility to **collapse**. The element will be hidden, and it will not take up any space in the layout.

```html live "v-c" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```

## Invisible

This example sets the visibility to **invisible**. The element will be hidden, but it will still take up space in the layout.

```html live "v-i" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```

## Visible

This example sets the visibility to **visible**. The element will be displayed and take up space in the layout.

```html live "v-v" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```
