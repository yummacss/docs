Controls the aspect ratio of an element.

<Utility
  data={[
    {
      className: "ar-auto",
      properties: ["aspect-ratio: auto;"],
    },
    {
      className: "ar-l",
      properties: ["aspect-ratio: 16/9;"],
    },
    {
      className: "ar-s",
      properties: ["aspect-ratio: 1/1;"],
    },
  ]}
/>

## Auto

This example sets the aspect ratio to **auto**. The element will maintain its default aspect ratio based on its content.

```html live "ar-auto" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```

## Landscape

This example sets the aspect ratio to **16/9**. The element will maintain a widescreen aspect ratio, commonly used for videos.

```html live "ar-l" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```

## Square

This example sets the aspect ratio to **1/1**. The element will maintain a square aspect ratio.

```html live "ar-s" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```
