<Class name="/clear" />

## Both

This example sets the clear property to **both**. The element will not allow any floating elements on either side.

```html live "cl-b" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

## Inline Start

This example sets the clear property to **inline-start**. The element will not allow any floating elements on the inline start side.

```html live "cl-is" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

## Inline End

This example sets the clear property to **inline-end**. The element will not allow any floating elements on the inline end side.

```html live "cl-ie" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

## Left

This example sets the clear property to **left**. The element will not allow any floating elements on the left side.

```html live "cl-l" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```



## None

This example sets the clear property to **none**. The element can have floating elements on both sides.

```html live "cl-none" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```
