Controls the element's stack order.

<Utility
  additionalClasses={[{ name: "auto", value: "auto" }]}
  classPrefix="zi-"
  incrementPrefix={10}
  incrementValue={10}
  propNames={["z-index"]}
  range={9}
/>

This example sets the z-index to **10**. The element will be positioned above elements with a lower z-index value.

```html live "zi-10" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

<Modifier
  utility="zi-10"
  prefixUtil="zi"
  modifiers={{
    hoverUtil: "zi-20",
    responsiveUtil: "zi-20",
  }}
/>
