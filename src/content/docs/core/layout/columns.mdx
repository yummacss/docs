Controls how many columns are in an element.

<Utility classPrefix="cols-" excludeZero propNames={["columns"]} incrementValue={1} range={16} />

This example sets the number of columns to **4**. The content will be displayed in a six-column layout.

```html live "cols-4" layout="/src/layouts/wide.astro"
<div class="cols-4">
  <p class="bg-indigo rad-1 ta-c tc-white p-4">A</p>
  <p class="bg-indigo rad-1 ta-c tc-white p-4">B</p>
  <p class="bg-indigo rad-1 ta-c tc-white p-4">C</p>
  <p class="bg-indigo rad-1 ta-c tc-white p-4">D</p>
</div>
```
