## Bottom

Controls the bottom placement of positioned elements.

<Utility
  additionalClasses={[{ name: "none", value: "none" }]}
  classPrefix="dir-b-"
  incrementValue={0.25}
  propNames={["bottom"]}
  range={16}
  unit="rem"
/>

```html live "dir-b-0" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```
