## 0

This example sets the letter spacing to **0em**. The letters will be spaced normally, with no additional spacing applied.

```html live "ls-0" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <p class="bg-white fs-sm ls-0 p-4 rad-1 ta-c tc-lead">The quick brown fox jumps over the lazy dog.</p>
</div>
```

## 1

This example sets the letter spacing to **-0.05em**. The letters will be slightly closer together than normal, creating a tighter appearance.

```html live "ls-1" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <p class="bg-white fs-sm ls-1 p-4 rad-1 ta-c tc-lead">The quick brown fox jumps over the lazy dog.</p>
</div>
```
