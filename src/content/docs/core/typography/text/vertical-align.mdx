## Bottom

This example sets the vertical alignment to **bottom**. The element will be aligned to the bottom of the line box.

```html live "va-b" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```

## Baseline

This example sets the vertical alignment to **baseline**. The element will be aligned to the baseline of the parent element.

```html live "va-ba" layout="/src/layouts/wide.astro"

```
