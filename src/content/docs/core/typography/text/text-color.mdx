## Indigo

This example sets the text color to **tc-indigo**. The text will be displayed in indigo color.

```html live "tc-indigo" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c tc-indigo">This text is indigo.</p>
```

## Light Indigo

This example sets the text color to **tc-l-indigo-4**. The text will be displayed in a lighter shade of indigo.

```html live "tc-l-indigo-4" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c tc-l-indigo-4">This text is indigo-4.</p>
```

## Dark Indigo

This example sets the text color to **tc-d-indigo-4**. The text will be displayed in a darker shade of indigo.

```html live "tc-d-indigo-4" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c tc-d-indigo-4">This text is indigo-8.</p>
```
