## Balance

This example sets the text wrap to **balance**. The text will be wrapped in a way that balances the line lengths, creating a more visually appealing layout.

```html live "tw-b" layout="/src/layouts/wide.astro"
<style>
  @import url("https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@release/dist/yumma.css");
</style>
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white d-g g-4 max-w-80 mx-auto p-4 rad-1">
    <div class="fs-xl fw-600 tw-b">The quick brown fox jumps over the lazy dog</div>
    <p class="fs-sm tc-gray-7">
      The quick brown fox jumps over the lazy dog. It is a well-known pangram, one that has gained significant popularity and has
      been utilized by many people. As a pangram, it contains all the letters of the alphabet, meeting the criteria for such a
      composition.
    </p>
  </div>
</div>
```

## No Wrap

This example sets the text wrap to **nowrap**. The text will not wrap to the next line, and it will overflow the container if it exceeds the width.

```html live "tw-n" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white d-g g-4 max-w-80 mx-auto p-4 rad-1">
    <div class="fs-xl fw-600 tw-nw">The quick brown fox jumps over the lazy dog</div>
    <p class="fs-sm tc-gray-7">
      The quick brown fox jumps over the lazy dog. It is a well-known pangram, one that has gained significant popularity and has
      been utilized by many people. As a pangram, it contains all the letters of the alphabet, meeting the criteria for such a
      composition.
    </p>
  </div>
</div>
```

## Wrap

This example sets the text wrap to **wrap**. The text will wrap to the next line when it reaches the end of the container.

```html live "tw-w" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white d-g g-4 max-w-80 mx-auto p-4 rad-1">
    <div class="fs-xl fw-600 tw-w">The quick brown fox jumps over the lazy dog</div>
    <p class="fs-sm tc-gray-7">
      The quick brown fox jumps over the lazy dog. It is a well-known pangram, one that has gained significant popularity and has
      been utilized by many people. As a pangram, it contains all the letters of the alphabet, meeting the criteria for such a
      composition.
    </p>
  </div>
</div>
```
