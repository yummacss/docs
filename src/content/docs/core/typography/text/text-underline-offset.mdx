## 0

This example sets the text underline offset to **0px**. The underline will be positioned directly under the text with no additional offset.

```html live "tuo-0" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <p class="bg-white fs-sm p-4 rad-1 ta-c tc-lead td-u tuo-0">The quick brown fox jumps over the lazy dog.</p>
</div>
```

