## None

This example sets the text decoration to **none**. The text will be displayed without any decoration, such as underlines or strikethroughs.

```html live "tdc-yellow" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-j tc-lead">
    The quick brown fox <span class="tdc-none tdl-u tdt-2">jumps</span> over the lazy dog, and the dog
    <span class="tdc-none tdl-u tdt-2">barks</span> loudly. However, the fox doesn't seem to mind. It continues to
    <span class="tdc-none tdl-u tdt-2">run</span> swiftly, while the dog
    <span class="tdc-none tdl-u tdt-2">chases</span> after it. In the end, the fox
    <span class="tdc-none tdl-u tdt-2">realises</span> that it can be friends with the dog, despite their differences.
  </div>
</div>
```

## Underline

This example sets the text decoration to **underline**. The text will be displayed with an underline, emphasizing the content.

```html live "tdc-yellow" "tdc-red" "tdc-teal" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-j tc-lead">
    The <span class="tdc-yellow tdl-u tdt-2">quicks</span> brown fox jumps over the lazy dog, and the dog barks loudly.
    <span class="tdc-teal tdl-u tdt-2">However</span>, the fox doesn't seem to mind. It continues to run
    <span class="tdc-teal tdl-u tdt-2">swiftly</span>, while the dog chases after it. In the end, the fox
    <span class="tdc-red tdl-u tdt-2">realises</span> that it can be friends with the dog, despite their differences.
  </div>
</div>
```
