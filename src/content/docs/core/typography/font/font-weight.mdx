Controls the font weight of an element.

<Utility
  classPrefix="fw-"
  excludeZero
  incrementPrefix={100}
  incrementValue={100}
  propNames={["font-weight"]}
  range={9}
/>

## 100
This example sets the font weight to **100**. This weight is very light and is typically used for fine text.

```html live "fw-100" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```
