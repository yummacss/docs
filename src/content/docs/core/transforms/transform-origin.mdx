## Bottom

This example sets the transform origin to **bottom**. The **t-o-b** utility defines the point around which transformations (such as rotations or scaling) are applied, positioning it at the bottom edge of the element.

```html live "t-o-b"
<div class="d-g g-16 gtc-1 sm:gtc-3">
  <div class="my-4 p-r">
    <div class="bg-indigo d-16 d-ib rad-1 t-o-b t-r-30"></div>
    <div class="o-20 bg-indigo d-16 d-ib l-0 p-a rad-1"></div>
  </div>

  <div class="my-4 p-r">
    <div class="bg-indigo d-16 d-ib rad-1 t-o-b t-r-45"></div>
    <div class="o-20 bg-indigo d-16 d-ib l-0 p-a rad-1"></div>
  </div>

  <div class="my-4 p-r">
    <div class="bg-indigo d-16 d-ib rad-1 t-o-b t-r-55"></div>
    <div class="o-20 bg-indigo d-16 d-ib l-0 p-a rad-1"></div>
  </div>
</div>
```
