<Class name="/transform" />

This example applies a **skew transformation** to the element. The **skew-12** utility skews the element by **12 degrees** along the X-axis, creating a slanted effect.

```html live "t-sk-12"
<style>
  @import url("https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@release/dist/yumma.css");
</style>
<div class="d-g g-16 gtc-1 sm:gtc-3">
  <div class="ai-c bg-indigo d-20 d-f jc-c rad-1 tc-white t-sk-12">A</div>
  <div class="ai-c bg-indigo d-20 d-f jc-c rad-1 tc-white t-sk-12">B</div>
  <div class="ai-c bg-indigo d-20 d-f jc-c rad-1 tc-white t-sk-12">C</div>
</div>
```
