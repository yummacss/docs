Controls the ability of the user to select text in an element.

<Utility
  data={[
    {
      className: "us-a",
      properties: ["user-select: all;"],
    },
    {
      className: "us-auto",
      properties: ["user-select: auto;"],
    },
    {
      className: "us-none",
      properties: ["user-select: none;"],
    },
    {
      className: "us-t",
      properties: ["user-select: text;"],
    },
  ]}
/>

## All

This example sets the user select to **all**. The user can select all text within the element.

<Message icon="text_select">Try selecting the sentence to see how the text highlighting works.</Message>

```html live "us-a" layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-c us-a">The quick brown fox jumps over the lazy dog. </div>
</div>
```

## Auto

This example sets the user select to **auto**. The user can select text as per the default behavior of the element.

<Message icon="text_select">Try selecting the sentence to see how the text highlighting works.</Message>

```html live layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-c us-auto">The quick brown fox jumps over the lazy dog.</div>
</div>
```

## None

This example sets the user select to **none**. The user cannot select any text within the element.

<Message icon="text_select">Try selecting the sentence to see how the text highlighting works.</Message>

```html live layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-c us-none">The quick brown fox jumps over the lazy dog.</div>
</div>
```

## Text

This example sets the user select to **text**. The user can select text within the element, while other elements remain unselectable.

<Message icon="text_select">Try selecting the sentence to see how the text highlighting works.</Message>

```html live layout="/src/layouts/wide.astro"
<div class="bg-l-indigo-6 p-4 rad-1">
  <div class="bg-white p-4 rad-1 ta-c us-t">The quick brown fox jumps over the lazy dog.</div>
</div>
```
