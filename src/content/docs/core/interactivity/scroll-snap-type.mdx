Controls the strictness of snap points in a snap container.

<Class name="/scroll-snap-type" />

## Both Mandatory

This example sets the scroll snap type to **both mandatory**. The scroll will snap in both directions (x and y) and will always snap to the nearest snap point.

<Message icon="mouse">Try scrolling through the image container to see how it behaves.</Message>

```html live "sst-b-m" layout="/src/layouts/wide.astro"
<div class="ovf-h p-r">
  <div class="d-f g-6 ovf-x-auto p-6 sst-b-m">
    <div class="fs-0 p-6 p-r sm-6 ssa-s">
```
