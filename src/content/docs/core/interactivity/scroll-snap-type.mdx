## Both Mandatory

This example sets the scroll snap type to **both mandatory**. The **sst-b-m** class enables snapping in both horizontal and vertical directions, requiring the scroll to stop at defined snap points.

<Message icon="mouse">Try scrolling through the image container to see how it behaves.</Message>

```html live "sst-b-m" layout="/src/layouts/wide.astro"
<div class="ovf-h p-r">
  <div class="d-f g-6 ovf-x-auto p-6 sst-b-m">
    <div class="fs-0 p-6 p-r sm-6 ssa-s">
      <img class="h-40 p-r rad-1 w-80 zi-10" src="https://picsum.photos/id/849/400/200" />
    </div>
    <div class="fs-0 p-6 p-r sm-6 ssa-s">
```

This example sets the scroll snap type to **none**. The **sst-none** class disables any snapping behavior, allowing for free scrolling without restrictions.
