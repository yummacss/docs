This example showcases the default browser **accent color** and a custom Indigo color. The custom accent will have a rich indigo shade.

```html live "ac-l-indigo-4"
<div class="d-g g-8 gtc-1 sm:gtc-2">
  <label><input class="mr-1" type="checkbox" checked />Default </label>
  <label><input class="ac-l-indigo-4 mr-1" type="checkbox" checked />Custom</label>
</div>
```
