Sets the accent color for the user interface controls generated by some elements.

```html live "ac-l-indigo-4"
<div class="d-f s-x-6">
  <label><input class="mr-1" type="checkbox" checked />Default</label>
  <label><input class="ac-l-indigo-4 mr-1" type="checkbox" checked />Custom</label>
</div>
```
