Controls the color of the visible marker where the next character typed will be inserted.

<Message icon="input_select">Try writing a sentence to see the caret color.</Message>

```html live "cc-indigo"
<div class="d-f s-x-6">
  <div class="d-f fd-c">

    <input class="mr-1" placeholder="Type something..." />
  </div>
  <div class="d-f fd-c">
    <label>Custom</label>
    <input class="cc-indigo mr-1" placeholder="Type something..." />
  </div>
</div>
```
