This example showcases the default browser **caret color** and a custom Indigo color. The custom caret will have a rich indigo shade.

<Decorator name="input_select">Try writing a sentence to see the caret color.</Decorator>

```html live "cc-indigo"
<div class="d-g g-16 gtc-1 sm:gtc-2">
  <div>
    <label class="fw-600">Default</label>
    <input class="b-s p-1 rad-1" placeholder="Type something..." />
  </div>

  <div>
    <label class="fw-600">Custom</label>
    <input class="b-s cc-indigo p-1 rad-1 " placeholder="Type something..." />
  </div>
</div>
```
