Controls the color of the visible marker where the next character typed will be inserted.

```html live "cc-indigo"
<div class="d-f fd-c">
  <label class="fw-600">Click to see the caret</label>
  <input class="cc-indigo max-w-65" placeholder="Search..." />
</div>
```
