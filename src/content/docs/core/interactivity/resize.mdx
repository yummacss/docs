Controls the way an item can be resized.

<Utility
  data={[
    {
      className: "r-b",
      properties: ["resize: both;"],
    },
    {
      className: "r-h",
      properties: ["resize: horizontal;"],
    },
    {
      className: "r-none",
      properties: ["resize: none;"],
    },
    {
      className: "r-v",
      properties: ["resize: vertical;"],
    },
  ]}
/>

## Both

The element can be resized both horizontally and vertically.

```html live
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>
  <textarea class="b-1 b-s max-w-100 rad-2 r-b" rows="4"></textarea>
</div>
```

## Horizontal

The element can be only be resized horizontally.

```html live
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>
  <textarea class="b-1 b-s max-w-100 rad-2 r-h" rows="4"></textarea>
</div>
```

## None

The element cannot be resized.

```html live
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>
  <textarea class="b-1 b-s max-w-100 rad-2 r-none" rows="4"></textarea>
</div>
```

## Vertical

The element can be only be resized vertically.

```html live
<div class="d-f fd-c">
  <label class="mb-2">Write a message:</label>
  <textarea class="b-1 b-s max-w-100 rad-2 r-v" rows="4"></textarea>
</div>
```
