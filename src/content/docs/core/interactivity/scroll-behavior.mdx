Controls the scroll behavior of an element.

<Class name="/scroll-behavior" />

## Auto

The scrolling box scrolls instantly.

```html "sb-auto"
<nav class="b-1 b-1 bb-0 bc-silver-2 d-b mx-auto p-1 rad-t-2 ta-c tc-white w-50">
  <a href="#month-1" class="d-ib fw-700 tc-lead td-none w-12">1</a>
  <a href="#month-2" class="d-ib fw-700 tc-lead td-none w-12">2</a>
  <a href="#month-3" class="d-ib fw-700 tc-lead td-none w-12">3</a>
</nav>
<div class="b-1 bc-silver-2 d-b h-52 mx-auto ovf-y-s rad-b-2 sb-auto ta-c w-50">
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-1">Jan</div>
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-2">Feb</div>
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-3">Mar</div>
</div>
```

## Smooth

The scrolling box scrolls smoothly.

```html "sb-s"
<nav class="b-1 b-1 bb-0 bc-silver-2 d-b mx-auto p-1 rad-t-2 ta-c tc-white w-50">
  <a href="#month-1" class="d-ib fw-700 tc-lead td-none w-12">1</a>
  <a href="#month-2" class="d-ib fw-700 tc-lead td-none w-12">2</a>
  <a href="#month-3" class="d-ib fw-700 tc-lead td-none w-12">3</a>
</nav>
<div class="b-1 bc-silver-2 d-b h-52 mx-auto ovf-y-s rad-b-2 sb-s ta-c w-50">
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-1">Jan</div>
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-2">Feb</div>
  <div class="ai-c d-f fs-5xl h-full jc-c" id="month-3">Mar</div>
</div>
```
