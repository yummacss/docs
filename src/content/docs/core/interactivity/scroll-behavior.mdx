<Class name="/scroll-behavior" />

## Auto

This example sets the scroll behavior to **auto**. The **sb-auto** class allows the default scrolling behavior, which is immediate and without any animation.

```html live "sb-auto" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this example.</p>
```

## Smooth

This example sets the scroll behavior to **smooth**. The **sb-s** class enables a smooth scrolling effect, providing a gradual transition when scrolling to a different part of the page.

```html live "sb-s" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this example.</p>
```

<Modifier
  utility="sb-auto"
  prefixUtil="sb"
  modifiers={{
    hoverUtil: "sb-s",
    responsiveUtil: "sb-s",
  }}
