## Always

This example sets the scroll snap stop to **always**. The **sss-a** class ensures that the scroll will always stop at the snap position.

<Decorator name="mouse">Try scrolling through the image container to see how it behaves.</Decorator>

```html live "sss-a" layout="/src/layouts/wide.astro"
<div class="ovf-h pb-4 p-r">
  <div class="d-f g-6 ovf-x-auto pb-4 sst-x-m">
    <div class="fs-0 p-r ssa-c">
      <div class="h-40 rad-1 stripes w-80"></div>
    </div>
    <div class="fs-0 p-r ssa-c sss-a">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/849/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-a">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/866/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-a">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/872/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-a">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/875/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-a">
      <div class="h-40 rad-1 stripes w-80"></div>
    </div>
  </div>
</div>
```

## Normal

This example sets the scroll snap stop to **normal**. The **sss-n** class allows the scroll to stop at the snap position only when the user scrolls slowly, providing a more flexible scrolling experience.

<Decorator name="mouse">Try scrolling through the image container to see how it behaves.</Decorator>

```html live "sss-n" layout="/src/layouts/wide.astro"
<div class="ovf-h pb-4 p-r">
  <div class="d-f g-6 ovf-x-auto pb-4 sst-x-m">
    <div class="fs-0 p-r ssa-c">
      <div class="h-40 rad-1 stripes w-80"></div>
    </div>
    <div class="fs-0 p-r ssa-c sss-n">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/849/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-n">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/866/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-n">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/872/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-n">
      <img class="h-40 rad-1 w-80" src="https://picsum.photos/id/875/400/200" />
    </div>
    <div class="fs-0 p-r ssa-c sss-n">
      <div class="h-40 rad-1 stripes w-80"></div>
    </div>
  </div>
</div>
```
