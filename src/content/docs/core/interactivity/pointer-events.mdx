    },
  ]}

## Auto

This example sets the pointer events to **auto**. The element can receive pointer events as normal.

```html live "pe-auto" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```

## None

This example sets the pointer events to **none**. The element will not receive any pointer events, making it unclickable.

```html live "pe-none" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're working on this example.</p>
```

<Modifier
  utility="pe-none"
  prefixUtil="pe"
  modifiers={{
    responsiveUtil: "pe-auto",
    hoverUtil: "pe-auto",
  }}
/>
