## Auto

This example sets the appearance to **auto**. The element will use the default styling provided by the browser.

```html live "a-auto" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```

## None

This example sets the appearance to **none**. The element will have no default styling.

```html live "a-none" layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We are currently working on this example.</p>
```

<Modifier
  utility="a-none"
  prefixUtil="a"
  modifiers={{
    hoverUtil: "a-auto",
    responsiveUtil: "a-auto",
  }}
/>
