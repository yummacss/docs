## Slice

This example sets the box decoration break to **slice**. The decoration will be applied to each box separately.

```html live layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

## Clone

This example sets the box decoration break to **clone**. The decoration will be applied to the entire width of the container.

```html live layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```
