Controls the repetition of the background image of an element.

<Class name="/background-repeat" />

## No Repeat

The background image will not repeat.

{/* prettier-ignore */}
```html live "br-nr"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-nr bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat

The background image will repeat both horizontally and vertically.

{/* prettier-ignore */}
```html live "br-r"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-r bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Round

The background image will scale to fit the element and repeat as necessary.

```html live
<div class="p-4 p-r rad-1 bs-lg">
  <div
    class="br-ro bp-c d-full p-r rad-1"
    style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat X

The background image will repeat horizontally.

{/* prettier-ignore */}
```html live
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-rx bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat Y

The background image will repeat vertically.

{/* prettier-ignore */}
```html live
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-ry bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Space

The background image will repeat, leaving space between each image.

{/* prettier-ignore */}
```html live
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-s bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```
