Controls the repetition of the background image of an element.

<Class name="/background-repeat" />

## No Repeat

This example sets the background repeat to **no-repeat**. The background image will not be repeated.

```html live "br-nr"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-nr bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat

This example sets the background repeat to **repeat**. The background image will be repeated both horizontally and vertically.

```html live "br-r"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-r bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Round

This example sets the background repeat to **round**. The background image will be repeated and resized to fit the area.

```html live "br-ro"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-ro bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat X

This example sets the background repeat to **repeat-x**. The background image will be repeated horizontally only.

```html live "br-rx"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-rx bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Repeat Y

This example sets the background repeat to **repeat-y**. The background image will be repeated vertically only.

```html live "br-ry"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-ry bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Space

The background image will repeat, leaving space between each image.

```html live "br-s"
<div class="p-4 p-r rad-1 bs-lg">
  <div class="br-s bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```
