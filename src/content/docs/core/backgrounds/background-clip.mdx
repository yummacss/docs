Controls the box around the background of an element.

<Class name="/background-clip" />

## Box Box

The background goes beyond the border.

```html live "bc-bb"
<div class="b-3 b-d bc-bb bc-d-indigo-4 bg-indigo dim-20 p-2 rad-1"></div>
```

## Content Box

The background goes all the way to the inside edge of the border.

```html live "bc-cb"
<div class="b-3 b-d bc-cb bc-d-indigo-4 bg-indigo dim-20 p-2 rad-1"></div>
```

## Padding Box

The background goes all the way to the edge of the content box.

```html live "bc-pb"
<div class="b-3 b-d bc-d-indigo-4 bc-pb bg-indigo dim-20 p-2 rad-1"></div>
```
