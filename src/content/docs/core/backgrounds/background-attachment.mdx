Controls how the background responds to scrolling.

<Class name="/background-attachment" />

## Fixed

This example sets the background image to **fixed**. The image will not move when the content is scrolled.

{/* prettier-ignore */}
```html live "ba-f"
<div class="ba-f bc-b bp-c bs-c h-32 ovf-y-s rad-1 w-full" style='background-image: url("https://picsum.photos/id/870/300/300");'>
  <p class="fs-3xl fw-600 p-6 ta-c tc-white">
    "The quick brown fox jumps over the lazy dog"
  </p>
</div>
```

## Local

This example sets the background image to **local**. The image will scroll with the content of the element.

{/* prettier-ignore */}
```html live "ba-l"
<div class="ba-l bc-b bp-c bs-c h-32 ovf-y-s rad-1 w-full" style='background-image: url("https://picsum.photos/id/870/300/300");'>
  <p class="fs-3xl fw-600 p-6 ta-c tc-white">
    "The quick brown fox jumps over the lazy dog"
  </p>
</div>
```

## Scroll

This example sets the background image to **scroll**. The image will move with the content of the page.

{/* prettier-ignore */}
```html live "ba-s"
<div class="ba-s bc-b bp-c bs-c h-32 ovf-y-s rad-1 w-full" style='background-image: url("https://picsum.photos/id/870/300/300");'>
  <p class="fs-3xl fw-600 p-6 ta-c tc-white">
    "The quick brown fox jumps over the lazy dog"
  </p>
</div>
```
