## Fixed

This example sets the background image to **fixed**. The image will not move when the content is scrolled.

<Decorator name="mouse">Try scrolling through the image to see how it behaves.</Decorator>

```html "ba-f"
<div class="o-auto ai-c ba-f bs-lg d-f ff-c fs-i h-40 jc-c p-8 rad-2 ta-c tc-white w-84" style="background-image: url('https://picsum.photos/600/400?image=106');">
  <h1 class="bf-b-md fs-xl fw-700 px-4 py-2 rad-2">Hi, Anne. It's finally springtime here on earth! I can't stand windy or cold days...</h1>
</div>
```

## Local

This example sets the background image to **local**. The image will scroll with the content of the element.

<Decorator name="mouse">Try scrolling through the image to see how it behaves.</Decorator>

```html live "ba-l"
<div class="o-auto ai-c ba-l bp-c bs-c bs-lg d-f ff-c fs-i h-40 jc-c p-8 rad-2 ta-c tc-white w-84" style="background-image: url('https://picsum.photos/600/400?image=106');">
  <h1 class="bf-b-md fs-xl fw-700 px-4 py-2 rad-2">Hi, Anne. It's finally springtime here on earth! I can't stand windy or cold days...</h1>
</div>
```

## Scroll

This example sets the background image to **scroll**. The image will move with the content of the page.

<Decorator name="mouse">Try scrolling through the image to see how it behaves.</Decorator>

```html live "ba-s"
<div class="o-auto ai-c ba-s bp-c bs-c bs-lg d-f ff-c fs-i h-40 jc-c p-8 rad-2 ta-c tc-white w-84" style="background-image: url('https://picsum.photos/600/400?image=106');">
  <h1 class="bf-b-md fs-xl fw-700 px-4 py-2 rad-2">Hi, Anne. It's finally springtime here on earth! I can't stand windy or cold days...</h1>
</div>
```
