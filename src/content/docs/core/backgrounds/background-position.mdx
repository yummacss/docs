Controls the position of the background image of an element.

<Class name="/background-position" />

## Bottom

The background image will be at the bottom of the element.

{/* prettier-ignore */}
```html live "bp-p"
<div class="d-30 p-r rad-2 bs-lg">
  <div class="bp-p d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Center

The background image will be positioned in the center of the element.

{/* prettier-ignore */}
```html live "bp-c"
<div class="d-30 p-r rad-2 bs-lg">
  <div class="bp-c d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```
