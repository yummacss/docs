## Bottom

This example sets the background position to **bottom**. The background image will be aligned to the bottom of the element.

```html live "bp-p"
<div class="d-30 p-r rad-1 bs-lg">
  <div class="bp-p d-full p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```
