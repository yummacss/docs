Controls the size of the background image of an element.

<Class name="/background-size" />

## Auto

The background image will retain its original size.

{/* prettier-ignore */}
```html live "bs-auto"
<div class="d-30 p-r rad-1 bs-lg">
  <div class="bs-auto bp-c h-30 p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Cover

The background image will scale to cover the entire element.

{/* prettier-ignore */}
```html live "bs-c"
<div class="d-30 p-r rad-1 bs-lg">
  <div class="bs-c bp-c h-30 p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```

## Contain

The background image will scale to fit within the element while maintaining its aspect ratio.

{/* prettier-ignore */}
```html live "bs-co"
<div class="d-30 p-r rad-1 bs-lg">
  <div class="bs-co bp-c h-30 p-r rad-1" style="background-image:url(https://picsum.photos/id/870/300/300)"></div>
</div>
```
