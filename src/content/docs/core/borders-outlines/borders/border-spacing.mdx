## Border Spacing

```html live
<table class="b-1 bc-silver bs-2 w-full">
  <tr>
    <td class="b-1 bc-silver bg-silver-1 p-4">A</td>
    <td class="b-1 bc-silver bg-silver-2 p-4">B</td>
  </tr>
  <tr>
    <td class="b-1 bc-silver bg-silver-3 p-4">C</td>
    <td class="b-1 bc-silver bg-silver-4 p-4">D</td>
  </tr>
</table>
```
