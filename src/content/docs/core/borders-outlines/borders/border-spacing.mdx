<Class name="/border-spacing" />

This example sets the border spacing to **1rem**. The space between the borders of adjacent table cells will be increased.

```html live "bs-4" layout="/src/layouts/wide.astro"
<table class="b-1 bc-silver bs-4 w-full">
  <tr>
    <td class="b-1 bc-silver bg-silver-1 p-4 ta-c">A</td>
    <td class="b-1 bc-silver bg-silver-1 p-4 ta-c">B</td>
  </tr>
  <tr>
    <td class="b-1 bc-silver bg-silver-1 p-4 ta-c">C</td>
    <td class="b-1 bc-silver bg-silver-1 p-4 ta-c">D</td>
  </tr>
</table>
```
