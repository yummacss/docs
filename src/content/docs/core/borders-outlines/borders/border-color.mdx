```html live "bc-indigo" "bc-l-indigo-4" "bc-d-indigo-2"
<div class="d-g g-16 gtc-1 sm:gtc-3">

  <div class="b-2 bc-indigo dim-16 rad-1"></div>
  <div class="b-2 bc-d-indigo-2 dim-16 rad-1"></div>
</div>
```
