## Collapse

This example sets the border collapse to **collapse**. The borders of the table will be merged together.

```html live "bc-c" layout="/src/layouts/wide.astro"
<table class="b-1 b-s bc-c bc-l-silver-1 w-full">
  <tr>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">A</td>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">B</td>
  </tr>
  <tr>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">C</td>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">D</td>
  </tr>
</table>
```

## Separate

This example sets the border collapse to **separate**. The borders of the table cells will remain distinct, allowing for spacing between them.

```html live "bc-s" layout="/src/layouts/wide.astro"
<table class="b-1 b-s bc-l-silver-1 bc- ta-cs w-full">
  <tr>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">A</td>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">B</td>
  </tr>
  <tr>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">C</td>
    <td class="b-1 b-s bc-l-silver-1 p-2 ta-c">D</td>
  </tr>
</table>
```
