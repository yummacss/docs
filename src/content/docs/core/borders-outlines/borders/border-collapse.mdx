Controls the collapsing or separation of table borders.

<Utility
  data={[
    {
      className: "bc-c",
      properties: ["border-collapse: collapse;"],
    },
    {
      className: "bc-s",
      properties: ["border-collapse: separate;"],
    },
  ]}
/>

## Collapse

The table borders will be merged.

```html live layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```

## Separate

Each table cell will have its own border.

```html live layout="/src/layouts/wide.astro"
<p class="fw-600 ta-c">We're still working on this one!</p>
```
