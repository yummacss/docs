Controls the radius of the borders of an element.

<Utility
  additionalClasses={[
    { name: "full", value: "100%" },
    { name: "half", value: "50%" },
  ]}
  classPrefix="rad-"
  incrementValue={0.25}
  propNames={["border-radius"]}
  range={8}
  unit="rem"
/>

```html live mark="rad-0" "rad-2" "rad-4" "rad-half"
<div class="d-g g-16 gtc-4">
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-0 tc-white">0rem</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-2 tc-white">0.5rem</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-4 tc-white">1rem</div>
  <div class="ai-c bg-indigo d-f dim-16 fw-700 jc-c rad-half tc-white">50%</div>
</div>
```

## Bottom Left Radius

Controls the radius of the border of the bottom left of an element.

