<Utility
  additionalClasses={[
    { name: "full", value: "100%" },
    { name: "half", value: "50%" },
  ]}
  classPrefix="rad-"
  incrementValue={0.25}
  propNames={["border-radius"]}
  range={8}
  unit="rem"
/>

This example showcases various border radius values: **0.25rem**, **0.5rem**, and **0.75rem**. The element will have slightly rounded corners, moderately rounded corners, and more pronounced rounded corners, respectively.

```html live "rad-1" "rad-2" "rad-3" "rad-half"
<div class="d-g g-16 gtc-1 sm:gtc-3">
  <div class="bg-indigo dim-16 rad-1"></div>
  <div class="bg-indigo dim-16 rad-2"></div>
  <div class="bg-indigo dim-16 rad-3"></div>
</div>
```

## Bottom Left Radius

This example sets the bottom left border radius to **0.5rem**. The element will have moderately rounded corners at the bottom left.

<Utility
  additionalClasses={[

  ]}
  classPrefix="rad-bl-"
  incrementValue={0.25}
  propNames={["border-bottom-left-radius"]}

