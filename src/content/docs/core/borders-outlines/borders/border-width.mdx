<Class name="border-width" />

This example showcases various border widths: **1px**, **2px**, **3px**, and **4px**. The element will have borders that are thin, moderately thick, thick, and very thick, respectively.

```html live "b-1" "b-2" "b-3" "b-4"
<div class="d-g g-16 gtc-1 sm:gtc-3">
  <div class="ai-c b-1 bc-indigo bg-white d-f d-16 jc-c tc-indigo">A</div>
  <div class="ai-c b-2 bc-indigo bg-white d-f d-16 jc-c tc-indigo">B</div>
  <div class="ai-c b-3 bc-indigo bg-white d-f d-16 jc-c tc-indigo">C</div>
</div>
```

## Border Bottom Width

Controls the width of the bottom border of an element.

<Class name="border-bottom-width" />

This example sets the bottom border width to **3px**. The element will have a thick bottom border.

```html live "bb-3"
<div class="ai-c bb-3 bc-indigo bg-white d-f d-16 jc-c tc-indigo">A</div>
```
