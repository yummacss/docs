<Utility classPrefix="b-" incrementValue={1} propNames={["border-width"]} range={8} unit="px" />

This example showcases various border widths: **1px**, **2px**, **3px**, and **4px**. The element will have borders that are thin, moderately thick, thick, and very thick, respectively.

```html "b-1" "b-2" "b-3" "b-4"
<div class="d-g g-16 gtc-1 sm:gtc-4">
  <div class="ai-c b-1 bc-indigo bg-white d-f dim-16 jc-c tc-indigo">1</div>
  <div class="ai-c b-2 bc-indigo bg-white d-f dim-16 jc-c tc-indigo">2</div>
  <div class="ai-c b-3 bc-indigo bg-white d-f dim-16 jc-c tc-indigo">3</div>
  <div class="ai-c b-4 bc-indigo bg-white d-f dim-16 jc-c tc-indigo">4</div>
</div>
```
