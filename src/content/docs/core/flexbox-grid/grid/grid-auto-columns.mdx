Controls the size of grid columns created implicitly.

<Utility
  data={[
    {
      className: "gac-min",
      properties: ["grid-auto-columns: min-content;"],
    },
    {
      className: "gac-max",
      properties: ["grid-auto-columns: max-content;"],
    },
    {
      className: "gac-auto",
      properties: ["grid-auto-columns: auto;"],
    },
  ]}
/>

## Min Content

Sets the size of implicitly created columns to the minimum content size.

```html live "gac-min"
<div class="ovf-auto rad-1">
  <div class="d-g gac-min g-4 gtr-auto">
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">A long piece of text</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">B</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">C</div>
  </div>
</div>
```

## Max Content

Sets the size of implicitly created columns to the maximum content size.

```html live "gac-max"
<div class="ovf-auto rad-1">
  <div class="d-g gac-max g-4 gtr-auto">
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">A long piece of text</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">B</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">C</div>
  </div>
</div>
```

## Auto

Sets the size of implicitly created columns to adjust automatically based on the available space.

```html live "gac-auto"
<div class="ovf-auto rad-1">
  <div class="d-g gac-auto g-4 gtr-auto">
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">A long piece of text</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">B</div>
    <div class="ai-c bg-d-indigo-4 d-f jc-c p-4 rad-1 tc-white">C</div>
  </div>
</div>
```
