Controls the spacing between columns in a grid layout.

<Utility
  classPrefix="cg-"
  incrementValue={0.25}
  propNames={["column-gap"]}
  range={16}
  unit="rem"
/>

Adjusts the column gap between grid or flex items using predefined values.

```html live "rg-4" layout="/src/layouts/wide.astro"
<div class="rg-4 cg-10 d-g gtc-3 p-4">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">E</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">F</div>
</div>
```
