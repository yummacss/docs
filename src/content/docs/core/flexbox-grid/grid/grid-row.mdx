Controls the size and position of elements across rows.

## Grid Row Span

Defines the row span of a grid item.

<Utility
  classPrefix="gr-s-"
  excludeZero
  incrementName="span %i / span %i"
  incrementValue={1}
  propNames={["grid-row"]}
  range={16}
/>

The `gr-s-*` utility specifies how many rows the grid item should span.

```html live "gr-s-2" layout="/src/layouts/wide.astro"
<div class="d-g g-4 gtc-3 h-75 stripes ta-c tc-white">
  <div class="bg-indigo gr-s-2 p-4 rad-1">A</div>
  <div class="bg-d-indigo-4 p-4 rad-1">B</div>
  <div class="bg-d-indigo-4 p-4 rad-1">C</div>
  <div class="bg-d-indigo-4 p-4 rad-1">D</div>
  <div class="bg-d-indigo-4 p-4 rad-1">E</div>
</div>
```

## Grid Row End

Specifies where the grid item ends on the grid rows.

<Utility classPrefix="gre-" excludeZero incrementValue={1} propNames={["grid-row-end"]} range={16} />

The `gre-*` utility determines the row line where the grid item should stop.

```html live "gre-3" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="d-g g-4 gaf-c gtr-3 rad-1 stripes ta-c tc-white">
    <div class="bg-indigo d-g gr-s-2 gre-3 p-4 rad-1">A</div>
    <div class="bg-d-indigo-4 d-g gre-4 grs-1 p-4 rad-1">B</div>
    <div class="bg-d-indigo-4 d-g gre-4 grs-1 p-4 rad-1">C</div>
  </div>
</div>
```

## Grid Row Start

Specifies where the grid item begins on the grid rows.

<Utility classPrefix="grs-" excludeZero incrementValue={1} propNames={["grid-row-start"]} range={16} />

The `grs-*` utility defines the row line from which the grid item starts.

```html live "grs-2" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="d-g g-4 gaf-c gtr-3 rad-1 stripes ta-c tc-white">
    <div class="bg-indigo d-g gr-s-2 grs-2 p-4 rad-1">A</div>
    <div class="bg-d-indigo-4 d-g gre-4 grs-1 p-4 rad-1">B</div>
    <div class="bg-d-indigo-4 d-g gre-4 grs-1 p-4 rad-1">C</div>
  </div>
</div>
```
