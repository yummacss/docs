## Column

Auto-places grid items in a column flow direction.

```html live "gaf-c" layout="/src/layouts/wide.astro"
<div class="p-r tc-white">
  <div class="stripes d-g g-4 gaf-c gtc-3 gtr-3 ta-c">
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1 ta-c">A</div>
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1 ta-c">B</div>
    <div class="bg-indigo p-4 rad-1 ta-c">C</div>
    <div class="bg-d-indigo-4 p-4 rad-1 ta-c">D</div>
    <div class="bg-d-indigo-4 p-4 rad-1 ta-c">E</div>
  </div>
</div>
```

## Column Dense

Auto-places grid items with a dense packing algorithm to fill gaps.

```html live "gaf-cd" layout="/src/layouts/wide.astro"
<div class="p-r tc-white">
  <div class="stripes d-g g-4 gaf-cd gtc-3 gtr-3 ta-c">
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1 ta-c">A</div>
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1 ta-c">B</div>
    <div class="bg-indigo p-4 rad-1 ta-c">C</div>
    <div class="bg-d-indigo-4 p-4 rad-1 ta-c">D</div>
    <div class="bg-d-indigo-4 p-4 rad-1 ta-c">E</div>
  </div>
</div>
```

## Dense

Combines dense placement with a row flow direction.

```html live "gaf-d" layout="/src/layouts/wide.astro"
<div class="stripes d-g gaf-d g-4 tc-white">
  <div class="bg-indigo p-4 rad-1 ta-c">A</div>
  <div class="bg-indigo p-4 rad-1 ta-c">B</div>
  <div class="bg-indigo p-4 rad-1 ta-c">C</div>
  <div class="bg-indigo p-4 rad-1 ta-c">D</div>
</div>
```
