Controls the automatic placement of grid elements.

<Utility
  data={[
    {
      className: "gaf-c",
      properties: ["grid-auto-flow: column;"],
    },
    {
      className: "gaf-d",
      properties: ["grid-auto-flow: dense;"],
    },
    {
      className: "gaf-dr",
      properties: ["grid-auto-flow: dense row;"],
    },
    {
      className: "gaf-r",
      properties: ["grid-auto-flow: row;"],
    },
    {
      className: "gaf-rd",
      properties: ["grid-auto-flow: row dense;"],
    },
  ]}
/>

## Column

Auto-places grid items in a column flow direction.

```html live "gaf-c" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r">
  <div class="stripes d-g g-4 gaf-c gtc-3 gtr-3 ta-c tc-white">
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1">A</div>
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1">B</div>
    <div class="bg-indigo p-4 rad-1">C</div>
    <div class="bg-d-indigo-4 p-4 rad-1">D</div>
    <div class="bg-d-indigo-4 p-4 rad-1">E</div>
  </div>
</div>
```

## Column Dense

Auto-places grid items with a dense packing algorithm to fill gaps.

```html live "gaf-cd" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r">
  <div class="stripes d-g g-4 gaf-cd gtc-3 gtr-3 ta-c tc-white">
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1">A</div>
    <div class="bg-d-indigo-4 gc-s-2 p-4 rad-1">B</div>
    <div class="bg-indigo p-4 rad-1">C</div>
    <div class="bg-d-indigo-4 p-4 rad-1">D</div>
    <div class="bg-d-indigo-4 p-4 rad-1">E</div>
  </div>
</div>
```

## Dense

Combines dense placement with a row flow direction.

```html live "gaf-d" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-4">
  <div class="stripes d-g gaf-d g-4">
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">A</div>
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">B</div>
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">C</div>
    <div class="ai-c bg-indigo d-f jc-c p-4 rad-1 tc-white">D</div>
  </div>
</div>
```
