Controls the growth of flex elements.

<Utility
  classPrefix="fg-"
  incrementValue={1}
  propNames={["flex-grow"]}
  range={8}
/>

Controls how much of the flex container's positive free space, if there is any, should be used for the flex item's main size.

```html live "fg-0" "fg-1" "fg-2" layout="/src/layouts/wide.astro"
<div class="d-f g-4 p-4">
  <div class="bg-indigo fg-0 p-4 rad-1 tc-white">A</div>
  <div class="bg-indigo fg-1 p-4 rad-1 tc-white">B</div>
  <div class="bg-indigo fg-2 p-4 rad-1 tc-white">C</div>
</div>
```
