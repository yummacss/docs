## Auto

This example sets the align self to **auto**. The item will use the default alignment based on the container's alignment properties.

```html live "ai-auto" layout="/src/layouts/wide.astro"
<div class="tc-white">
  <div class="ai-s stripes d-f g-4 h-24 rad-1 w-full">
    <div class="ai-c bg-d-indigo-4 d-f f-1 jc-c p-4 rad-1">A</div>
    <div class="ai-c as-auto bg-indigo d-f f-1 jc-c p-4 rad-1">B</div>
    <div class="ai-c bg-d-indigo-4 d-f f-1 jc-c p-4 rad-1">C</div>
  </div>
</div>
```
