## Center

Aligns flex items at the center of the container.

```html live "jc-c" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-c tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Flex End

Aligns flex items to the end of the container.

```html live "jc-fe" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-fe tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Flex Start

Aligns flex items to the start of the container.

```html live "jc-fs" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-fs tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Normal

Resets the alignment to its default state.

```html live "jc-n" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-n tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Stretch

Stretches flex items across the container (default for some cases).

```html live "jc-s" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 rad-1 tc-white">
  <div class="stripes d-g g-4 gaf-c jc-s rad-1">
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">A</div>
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">B</div>
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">C</div>
  </div>
</div>
```

## Space Between

Distributes flex items with space between them.

```html live "jc-sb" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-sb tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```

## Space Around

Distributes flex items with space around them.

```html live "jc-sa" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 jc-sa tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
```

## Space Evenly

Distributes flex items with equal space between, before, and after them.

```html live "jc-se" layout="/src/layouts/wide.astro"
<div class="stripes d-f g-4 gaf-c jc-se tc-white">
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f dim-14 jc-c p-4 rad-1">C</div>
</div>
```
