## Baseline

Each line will align itself along the baseline of the container.

```html live "ac-b" layout="/src/layouts/wide.astro"
<div class="ac-b stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Center

Each line will align itself in the center of the cross axis.

```html live "ac-c" layout="/src/layouts/wide.astro"
<div class="ac-c stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Flex End

Each line will fill the space it needs and move towards the end of the cross axis.

```html live "ac-fe" layout="/src/layouts/wide.astro"
<div class="ac-fe stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Flex Start

Each line will fill the space it needs and move towards the start of the cross axis.

```html live "ac-fs" layout="/src/layouts/wide.astro"
<div class="ac-fs stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>

  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Normal

Each line will align in the default position along the cross axis.

```html live "ac-n" layout="/src/layouts/wide.astro"
<div class="ac-n stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Stretch

Each line will take up as much space as it can in the remaining space.

```html live "ac-s" layout="/src/layouts/wide.astro"
<div class="ac-s stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Space Around

Each line fills the space it needs. The remaining space will be distributed equally around the lines.

```html live "ac-sa" layout="/src/layouts/wide.astro"
<div class="ac-sa stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Space Between

Each line fills its space. The rest of the space will be between the lines.

```html live "ac-sb" layout="/src/layouts/wide.astro"
<div class="ac-sb stripes d-g g-4 gtc-3 h-64 tc-white ta-c w-full">
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">C</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">D</div>
  <div class="ai-c bg-indigo d-f jc-c p-4 rad-1">E</div>
</div>
```

## Space Evenly

Each line fills its space, and the remaining space is distributed evenly between and around the lines.
