<Utility classPrefix="fs-" incrementValue={1} propNames={["flex-shrink"]} range={8} />

Controls how much the flex item should shrink relative to the rest of the items in the flex container when there isn't enough space.

```html live "fs-0" "fs-1" "fs-2" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-3">
  <div class="stripes d-f g-4 rad-1 tc-white">
    <div class="ai-c bg-l-indigo-4 d-f dim-14 f-none jc-c p-4 rad-1">A</div>
    <div class="ai-c bg-indigo d-f fs-1 h-14 jc-c p-4 rad-1 w-64">B</div>
    <div class="ai-c bg-l-indigo-4 d-f dim-14 f-none jc-c p-4 rad-1">C</div>
  </div>
</div>
```
