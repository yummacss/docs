Use the `fb-*` utilities to control the initial size of a flex item before it grows or shrinks. These classes help visualize how flex-basis interacts with available space in a flex container.

```html live "fb-0" "fb-50" "fb-auto" "fb-full" layout="/src/layouts/wide.astro"
<div class="stripes tc-white">
  <div class="d-f g-4 h-24 w-full">
    <div class="ai-c bg-l-indigo-4 d-f fb-0 jc-c p-4 rad-1">A</div>
    <div class="ai-c bg-indigo d-f fb-50 jc-c p-4 rad-1">B</div>
    <div class="ai-c bg-l-indigo-4 d-f fb-full jc-c p-4 rad-1">C</div>
  </div>
</div>
```
