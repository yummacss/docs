Controls the alignment of grid elements relative to their inline axis.

<Utility
  data={[
    {
      className: "ji-c",
      properties: ["justify-items: center;"],
    },
    {
      className: "ji-e",
      properties: ["justify-items: end;"],
    },
    {
      className: "ji-s",
      properties: ["justify-items: start;"],
    },
    {
      className: "ji-st",
      properties: ["justify-items: stretch;"],
    },
  ]}
/>

Controls the alignment of grid items along the inline axis within their grid areas.

## Center

Aligns grid items to the center of their grid area.

```html live "ji-c" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="stripes d-g g-4 gtc-3 ji-c tc-white">
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">A</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">B</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">C</div>
  </div>
</div>
```

## End

Aligns grid items to the end of their grid area.

```html live "ji-e" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="stripes d-g g-4 gtc-3 ji-e tc-white">
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">A</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">B</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">C</div>
  </div>
</div>
```

## Start

Aligns grid items to the start of their grid area.

```html live "ji-s" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="stripes d-g g-4 gtc-3 ji-s tc-white">
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">A</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">B</div>
    <div class="ai-c bg-indigo d-f dim-14 jc-c rad-1">C</div>
  </div>
</div>
```

## Stretch

Stretches grid items to fill their grid area.

```html live "ji-st" layout="/src/layouts/wide.astro"
<div class="ovf-auto p-8 p-r rad-1">
  <div class="stripes d-g g-4 gtc-3 ji-st tc-white">
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">A</div>
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">B</div>
    <div class="ai-c bg-indigo d-f h-14 jc-c rad-1">C</div>
  </div>
</div>
```
