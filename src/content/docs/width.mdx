## Max Width

Controls the maximum width of an element.

<Baseline path="css.properties.max-width" />

<ApiTable category="boxModel" name="max-width" />

The **max-w-64** utility sets the maximum width to **16rem**, ensuring that the element does not exceed this width regardless of the content inside.

<Note icon="size">Try changing the horizontal size of the box to see how wide it can be at the maximum size.</Note>

## Using percentages

This example showcases various width values: `100%` and `50%`.

- The **w-full** utility sets the width to `100%`, allowing the element to take up the full width of its container.
- The **w-half** utility sets the width to `50%`, allowing the element to occupy half the width of its container.

<Preview>
  <div className="b-3 b-d bc-indigo d-g p-2 rg-8 w-64">
    <div className="bg-indigo p-2 ta-c tc-white w-full">100%</div>
    <div className="bg-indigo p-2 ta-c tc-white w-half">50%</div>
  </div>
</Preview>

```html "w-full" "w-half"
<div class="b-3 b-d bc-indigo d-g p-2 rg-8 w-64">
  <div class="bg-indigo p-2 ta-c tc-white w-full">100%</div>
  <div class="bg-indigo p-2 ta-c tc-white w-half">50%</div>
</div>
```
