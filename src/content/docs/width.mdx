### Max Width

Control an element's maximum width.

<Baseline path="css.properties.max-width" />

<Reference category="boxModel" name="max-width" />

The `max-w-64` utility limits an element's width to `16 rem`.

<Hint icon="arrows_horizontal">Try changing the horizontal size of the box to see how wide it can be at the maximum size.</Hint>

<Preview>
  <div className="ai-c bg-indigo d-f jc-c max-w-64 min-w-16 o-auto p-6 r-h c-white w-32">64</div>
</Preview>

```html "max-w-64"
<div class="ai-c bg-indigo d-f jc-c max-w-64 min-w-16 o-auto p-6 r-h c-white w-32">64</div>
```

### Min Width

Control an element's minimum width.

<Baseline path="css.properties.min-width" />

<Reference category="boxModel" name="min-width" />

The `min-w-16` utility ensures an element's width is at least `4 rem`.

<Hint icon="arrows_horizontal">Try changing the horizontal size of the box to see how wide it can be at the minimum size.</Hint>

<Preview>
  <div className="ai-c bg-indigo d-f jc-c max-w-64 min-w-16 o-auto p-6 r-h c-white w-32">32</div>
</Preview>

```html "min-w-16"
<div class="ai-c bg-indigo d-f jc-c max-w-64 min-w-16 o-auto p-6 r-h c-white w-32">32</div>
```
