## Usage

### Show

Use the `ec-s` utility to show borders & backgrounds for empty cells.

```html "ec-s"
<table class="ec-s ...">
  ...
</table>
```

### Hide

Use the `ec-h` utility to hide borders & backgrounds for empty cells.

```html "ec-h"
<table class="ec-h ...">
  ...
</table>
```
