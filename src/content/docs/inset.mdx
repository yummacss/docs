## Usage

### Inset

Control the placement of a positioned element in all directions.

```html "i-0"
<div class="p-r d-18 sm:d-32 c-white">
  <div class="p-a i-0 b-1"></div>
  <div class="p-a i-0 d-f ai-c jc-c bg-indigo p-4">A</div>
</div>
```

### Inset X

<Reference category="positioning" name="inset-x" />

Control the placement of a positioned element in the horizontal direction.

```html "ix-0"
<div class="p-r d-18 sm:d-32 c-white">
  <div class="p-a i-0 b-1"></div>
  <div class="p-a t-0 ix-0 d-f h-12 ai-c jc-c bg-indigo p-4">A</div>
</div>
```

### Inset Y

<Reference category="positioning" name="inset-y" />

Control the placement of a positioned element in the vertical direction.

```html "iy-0"
<div class="p-r d-18 sm:d-32 c-white">
  <div class="p-a i-0 b-1"></div>
  <div class="p-a iy-0 l-0 d-f w-12 ai-c jc-c bg-indigo p-4">A</div>
</div>
```
