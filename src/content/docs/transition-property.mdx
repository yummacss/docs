## Usage

```html
<div class="tp-a ...">All</div>
<div class="tp-bs ...">Box Shadow</div>
<div class="tp-c ...">Colors</div>
<div class="tp-none ...">None</div>
<div class="tp-o ...">Opacity</div>
<div class="tp-t ...">Transform</div>
```
