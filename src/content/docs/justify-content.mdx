## Normal

> Initial value

The items will be aligned according to the default alignment of the container.

  WIP
</Preview>

```html "jc-n" layout="/src/layouts/inline.astro"
<div class="d-f g-4 jc-n tc-white" id="area">
  <div class="ai-c bg-indigo d-f d-14 jc-c p-4 rad-1">A</div>
  <div class="ai-c bg-indigo d-f d-14 jc-c p-4 rad-1">B</div>
  <div class="ai-c bg-indigo d-f d-14 jc-c p-4 rad-1">C</div>
</div>
```

## Stretch

The items will stretch to fill the available space along the main axis.

  WIP
</Preview>

