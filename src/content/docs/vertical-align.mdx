## Usage

```html
<div class="ws-n ...">Normal</div>
<div class="ws-bs ...">Break Spaces</div>
<div class="ws-nw ...">No Wrap</div>
<div class="ws-p ...">Pre</div>
<div class="ws-pl ...">Pre-Line</div>
<div class="ws-pw ...">Pre-Wrap</div>
```
