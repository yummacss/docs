## Creating a new project

To create a new Astro project, you need to run the [Astro command](https://docs.astro.build/en/install-and-setup/) in your terminal.

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.{astro,ts,tsx}"],
      output: "./src/styles/global.css",
      buildOptions: {
        reset: true,
        minify: false,
      }
    };
    ```
  </Step>

  <Step title="Build styles">
    You can now start generating your CSS with the [`build`](/docs/foundations/config/#build) or [`watch`](/docs/foundations/config/#watch) command.

    ```bash
    pnpm yummacss build
    ```
  </Step>

  <Step title="Done!">
    You're all set to start using Yumma CSS utilities in your project.

    ```astro title="pages/index.astro" mark={3-5}
    ---
    import Layout from '../layouts/Layout.astro';
    ---

    <Layout title="Welcome to Astro.">
      <div class="bg-indigo-12 d-g h-dvh pi-c tc-white">
        <h1 class="fs-3xl fw-500">Yumma CSS ⚙️ Astro</h1>
      </div>
    </Layout>
    ```
  </Step>
</Stepper>
