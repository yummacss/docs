## Creating a new project

To create a new Astro project, you need run the [Astro CLI Wizard](https://docs.astro.build/en/install-and-setup/#install-from-the-cli-wizard) in your terminal.

<Steps>

    1. **Install Yumma CSS:**

        Install the Yumma CSS Library using a package manager.

        ```bash title="Installing dependencies..." frame="terminal"
        npm install yummacss@latest
        ```

    2. **Add** `yummacss` **in your `Layout.astro` file:**

        ```astro title="layouts/Layout.astro" mark={2}
        <style is:global>
            @import "/node_modules/yummacss/dist/yumma.min.css";
        </style>
        ```

    3. **Start development server**

        Start your application using `npm run dev`.

        ```bash title="Starting development server..."
        npm run dev
        ```

    4. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        ```astro title="pages/index.astro" mark={6,7,8}
        ---
        import Layout from '../layouts/Layout.astro';
        ---

        <Layout title="Welcome to Astro.">
            <div class="h-1/1 ins">
                <h1 class="fs-xxl fw-500 tc-pink">Yumma CSS + Astro</h1>
            </div>
        </Layout>
        ```

</Steps>
