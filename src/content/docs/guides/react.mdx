## Creating a new project

To create a new React project, you need to run the [React command](https://react.dev/learn/build-a-react-app-from-scratch#vite) in your terminal.

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/index.css",
      buildOptions: {
        reset: true,
        minify: false,
      }
    };
    ```
  </Step>

  <Step title="Build styles">
    You can now start generating your CSS with the [`build`](/docs/foundations/config/#build) or [`watch`](/docs/foundations/config/#watch) command.

    ```bash
    pnpm yummacss build
    ```
  </Step>

  <Step title="Done!">
    You're all set to start using Yumma CSS utilities in your project.

    ```tsx title="App.tsx" mark={3-5}
    function App() {
      return (
        <div className="bg-indigo-12 d-g h-dvh pi-c tc-white">
          <h1 className="fs-3xl fw-500">Yumma CSS ⚙️ React</h1>
        </div>
      );
    }

    export default App;
    ```
  </Step>
</Stepper>
