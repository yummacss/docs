## Creating a new project

To create a new Svelte project, you need to run the [Svelte command](https://svelte.dev/docs/cli/sv-create) in your terminal.

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default {
      source: ["./src/**/*.svelte"],
      output: "./src/global.css",
      buildOptions: {
        reset: true,
        minify: false,
      }
    };
    ```
  </Step>

  <Step title="Build styles">
    You can now start generating your CSS with the [`build`](/docs/foundations/config/#build) or [`watch`](/docs/foundations/config/#watch) command.

    ```bash
    pnpm yummacss build
    ```
  </Step>

  <Step title="Done!">
    You're all set to start using Yumma CSS utilities in your project.

    ```svelte title="routes/+page.svelte" mark={1}
    <div class="bg-indigo-12 d-g h-dvh pi-c tc-white">
      <h1 class="fs-3xl fw-500">Yumma CSS ⚙️ Svelte</h1>
    </div>
    ```
  </Step>
</Stepper>
