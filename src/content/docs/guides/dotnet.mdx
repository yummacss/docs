<Steps>

    1. **Navigate to your project:**

        Navigate to the solution your project:

        <Code code="cd project-name\solution-name" lang="bash" title="Terminal" />

    2. **Install Yumma CSS:**

        Install the Yumma CSS Library using [LibMan](https://learn.microsoft.com/en-us/aspnet/core/client-side/libman/libman-cli?view=aspnetcore-8.0).

        <Code code="libman install yummacss@latest --provider jsdelivr --destination lib/yummacss" lang="bash" title="Installing dependencies..." />

        <Aside type="note" title="Install without LibMan">
            You can also add `yummacss` to your project by right-clicking your project in **Solution Explorer** and selecting **Add** > **Client-Side Library**.
        </Aside>

        Once you've installed it, you should see the `yummacss` folder inside your `/lib` folder.

        <FileTree>
            - Properties/
            - **wwwroot**
                - css
                    - site.css
                - js
                    - site.js
                - lib/
                    - **yummacss/** generates this
                - favicon.ico
            - Controllers
                - HomeController.cs
            - Models
                - ErrorViewModel.cs
            - Views
                - Home/
                - Shared/
            - appsettings.json
            - libman.json
            - Program.cs
        </FileTree>

    3. **Include** `yummacss` **in your main CSS file:**

        <Code code={inCSS} lang="css" mark={[2]} title="css/site.css" />

    4. **Start development server**

        You can also press the `F5` key to access the development server.

    5. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        <Code code={onPage} lang="html" mark={[1,2,3]} title="Home/Index.cshtml" />

</Steps>
