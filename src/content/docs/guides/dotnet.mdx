<Steps>

    1. **Navigate to your project:**

        Navigate to the solution your project:

        ```bash title="Terminal"
        cd project-name\example
        ```

    2. **Install Yumma CSS:**

        Install the Yumma CSS Library using [LibMan](https://learn.microsoft.com/en-us/aspnet/core/client-side/libman/libman-cli?view=aspnetcore-8.0).

        ```bash title="Installing dependencies..."
        libman install yummacss@latest --provider jsdelivr --destination lib/yummacss
        ```

    3. **Include** `yummacss` **in your main CSS file:**

        In this case, we're importing the minified version of Yumma CSS.

        ```css title="css/site.css" mark={1}
        @import "/lib/yummacss/dist/yumma.min.css";
        ```

    4. **Start development server**

        You can also press the **F5** key to access the development server.

    5. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        ```html title="Home/Index.cshtml" mark={6}
        @model IndexModel
        @{
            ViewData["Title"] = "Home Page";
        }

        <h1 class="fs-sm fw-700 ta-c tc-indigo">Yumma CSS 🤝 .NET</h1>
        ```

</Steps>
