## Creating a new project

To create a new Angular project, you need to have the [Angular CLI](https://angular.dev/installation#install-angular-cli) installed on your machine.

<Steps>

    1. **Install Yumma CSS:**

        Install the Yumma CSS Library using a package manager.

        ```bash title="Installing dependencies..." frame="terminal"
        npm install yummacss@latest
        ```

    2. **Include** `yummacss` **in your main CSS file:**

        In this case, we're importing the minified version of Yumma CSS.

        ```css title="src/globals.css" mark={1}
        @import "../node_modules/yummacss/dist/yumma.min.css";
        ```

    3. **Start development server**

        To see the changes, start the development server.

        ```bash title="Starting development server..."
        npm run start
        ```

    4. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        ```html title="src/app/app.component.html" mark={1}
        <h1 class="fs-sm fw-700 ta-c tc-indigo">Yumma CSS 🤝 Angular</h1>
        ```

</Steps>
