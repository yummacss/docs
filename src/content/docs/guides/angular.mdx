How to add Yumma CSS to your project with Angular.

<br />

export const inCSS = guide.angular.inCSS;
export const onPage = guide.angular.onPage;

<Steps>

    1. **Install Yumma CSS:**

        Install the Yumma CSS Library using a package manager.

        <Code code="npm install yummacss@latest" lang="bash" title="Installing dependencies..." />

    2. **Include** `yummacss` **in your main CSS file:**

        <Code code={inCSS} lang="css" mark={[2]} title="src/styles.css" />

    3. **Start development server**

        <Code code="npm start" lang="bash" title="Terminal" />

    4. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        <Code code={onPage} lang="html" mark={[1, 2, 3]} title="src/app/app.component.html" />

</Steps>
