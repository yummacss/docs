## Creating a new project

To create a new Next.js project, you need to run the [create next app command](https://nextjs.org/docs/app/getting-started/installation) in your terminal.

<Stepper>
  <Step title="Install Yumma CSS">
    ```bash
    pnpm add yummacss -D
    ```
  </Step>

  <Step title="Initialize configuration">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Set up configuration">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={2-3}
    export default = {
      source: ["./app/**/*.{ts,tsx}"],
      output: "./app/globals.css",
      buildOptions: {
        reset: true,
        minify: false,
      }
    };
    ```
  </Step>

  <Step title="Build styles">
    You can now start generating your CSS with the [`build`](/docs/foundations/config/#build) or [`watch`](/docs/foundations/config/#watch) command.

    ```bash
    pnpm yummacss build
    ```
  </Step>

  <Step title="Done!">
    You're all set to start using Yumma CSS utilities in your project.

    ```tsx title="page.tsx" mark={3-5}
    export default function Home() {
      return (
        <div className="d-f ai-c jc-c h-screen">
          <h1 className="fs-4xl fw-700">Hello, Yumma CSS!</h1>
        </div>
      );
    }
    ```
  </Step>
</Stepper>
