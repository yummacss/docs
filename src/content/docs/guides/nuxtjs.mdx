How to add Yumma CSS to your project with Nuxt.js.

<br />

export const onCSS = guide.nuxtjs.onCSS;
export const onConfig = guide.nuxtjs.onConfig;
export const onPage = guide.nuxtjs.onPage;

<Steps>

    1. **Install Yumma CSS:**

        Install the Yumma CSS Library using a package manager.

        <Code code="npm i yummacss@latest" lang="bash" title="Installing dependencies..." />

    2. **Add** `yummacss` **to your main CSS file:**

        Create a new `assets/global.css` file.

        <Code code={onCSS} lang="css" mark={[2]} title="assets/global.css" />

    3. **Import the CSS file**

        Add the new CSS file to `nuxt.config.ts`.

        <Code code={onConfig} lang="ts" mark={4} title="nuxt.config.ts" />

    4. **Start development server**

        Start your application using `npm run dev`.

        <Code code="npm run dev" lang="bash" title="Terminal" />

    4. **Use Yumma CSS in your project:**

        You're all set to start using Yumma CSS utility classes in your project.

        <Code code={onPage} lang="vue" mark={[2,3,4]} title="app.vue" />

</Steps>
