<Steps>

    1. **Install Yumma CSS:**

        Install the Yumma CSS Library using a package manager.

        ```bash title="Installing dependencies..." frame="terminal"
        npm install yummacss@latest
        ```

    2. **Include** `yummacss` **in your main CSS file:**

        In this case, we're importing the minified version of Yumma CSS.

        Create a new `assets/global.css` file.

        ```css title="assets/global.css" mark={1}
        @import "yummacss/dist/yumma.min.css";
        ```

    3. **Import the CSS file**

        Add the new CSS file to `nuxt.config.ts`.

        ```ts title="nuxt.config.ts" mark={4}
        export default defineNuxtConfig({
            compatibilityDate: "2024-04-03",
            devtools: { enabled: true },
            css: ["/assets/global.css"],
        });
        ```
    4. **Start development server**

        Start your application using `npm run dev`.

        ```bash title="Starting development server..."
        npm run dev
        ```

    4. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        ```html title="app.vue" mark={2}
        <template>
            <h1 class="fs-sm fw-700 ta-c tc-indigo">Yumma CSS 🤝 Nuxt.js</h1>
        </template>
        ```

</Steps>
