## Creating a new project

To create a new Nuxt.js project, you need run the [Nuxt.js command](https://nuxt.com/docs/getting-started/installation#new-project) in your terminal.

<Steps>

    1. **Install Yumma CSS**

        Install `yummacss` using a package manager.

        <PackageManagers pkg="yummacss" title="Installing dependencies..." />

        <br />

    2. **Initialize configuration**

            Create a configuration file in your project.

            <PackageManagers pkg="yummacss init" title="Terminal" type="dlx" />

    3. **Set up configuration**

        Specify the locations of all your project files in the config file.

        ```js title="yumma.config.js" mark={2-3}
        export default {
            source: ["./app.vue"],
            output: "./assets/global.css",
            buildOptions: {
                reset: true,
                minify: false,
            }
        };
        ```

    4. **Build styles**

        You can now start generating your CSS with the [`build`](/docs/first-steps#build-command) or [`watch`](/docs/first-steps#watch-command) command.

        <PackageManagers pkg="yummacss build" title="Terminal" type="dlx" />

    5. **Done!**

        You're all set to start using Yumma CSS utility classes in your project.

        ```html title="app.vue" mark={2-4}
        <template>
            <div class="bg-indigo-12 d-g h-dvh pi-c tc-white">
                <h1 class="fs-3xl fw-500">Yumma CSS ⚙️ Nuxt.js</h1>
            </div>
        </template>
        ```

</Steps>
