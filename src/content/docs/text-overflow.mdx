## Clip

> Initial value

The text will be clipped at the edge of the container without any indication that it has been cut off.

<Preview>
  <p className="fs-lg o-h c-slate to-c w-35 ws-nw">Pneumonoultramicroscopicsilicovolcanoconiosis</p>
</Preview>

```html "to-c"
<p class="fs-lg o-h c-slate to-c w-35 ws-nw">Pneumonoultramicroscopicsilicovolcanoconiosis</p>
```

## Ellipsis

The text will be truncated with an ellipsis at the end to indicate that there is more content that is not visible.

<Preview>
  <p className="fs-lg o-h c-slate to-e w-35 ws-nw">Pneumonoultramicroscopicsilicovolcanoconiosis</p>
</Preview>

```html "to-e"
<p class="fs-lg o-h c-slate to-e w-35 ws-nw">Pneumonoultramicroscopicsilicovolcanoconiosis</p>
```
