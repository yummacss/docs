## Usage

### Static

Position the element according to the normal document flow.

<Preview variant="inline">
  <div className="c-white">
    <div className="bg-indigo p-4 p-s">Static Position</div>
    <div className="bg-indigo-2 p-4 c-slate">Another Element</div>
  </div>
</Preview>

```html "p-s"
<div class="c-white">
  <div class="bg-indigo p-4 p-s">Static Position</div>
  <div class="bg-indigo-2 p-4 c-slate">Another Element</div>
</div>
```

### Absolute

Position the element relative to its nearest positioned ancestor.

<Preview variant="inline">
  <div className="bg-indigo-2 h-32 p-4 p-r c-white">
    <div className="bg-indigo l-4 t-4 p-4 p-a">Absolute Position</div>
  </div>
</Preview>

```html "p-a"
<div class="bg-indigo-2 h-32 p-4 p-r c-white">
  <div class="bg-indigo l-4 t-4 p-4 p-a">Absolute Position</div>
</div>
```

### Fixed

Position the element relative to the viewport.

```html "p-f"
<div class="c-white">
  <div class="bg-indigo r-4 t-4 p-4 p-f">Fixed Position</div>
  <div class="bg-indigo-2 h-s p-4 c-white">Scroll down to see the fixed element.</div>
</div>
```

### Relative

Position the element relative to its normal position.

<Preview variant="inline">
  <div className="c-white">
    <div className="bg-indigo l-4 t-4 p-4 p-r">Relative Position</div>
    <div className="bg-indigo-2 p-4 c-slate">Another Element</div>
  </div>
</Preview>

```html "p-r"
<div class="c-white">
  <div class="bg-indigo l-4 t-4 p-4 p-r">Relative Position</div>
  <div class="bg-indigo-2 p-4 c-slate">Another Element</div>
</div>
```

    <div className="bg-indigo-2 h-96 p-4 c-slate">Scroll down to see the sticky behavior.</div>
  </div>
</Preview>

```html "p-st"
<div class="b-1 h-64 o-y-s c-white">
  <div class="bg-indigo t-0 p-4 p-st">Sticky Position</div>

```
