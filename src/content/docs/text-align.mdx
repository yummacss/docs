## Start

> Initial value

The text will be aligned to the start of the container, which is typically side-to-right languages.

<Preview className="bg-white">
  <div className="fs-md ta-s tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-s"
<div class="fs-md ta-s tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Center

The text will be centered within its container.

<Preview className="bg-white">
  <div className="fs-md ta-c tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-c"
<div class="fs-md ta-c tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## End

The text will be aligned to the end of the container, which is typically the right side-to-right languages.

<Preview className="bg-white">
  <div className="fs-md ta-e tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-e"
<div class="fs-md ta-e tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Justify

The text will be stretched to align with both and right edges of the container, creating a clean look.

<Preview className="bg-white">
  <div className="fs-md ta-j tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-j"
<div class="fs-md ta-j tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Justify All

The text will be justified, including the last line, which will also stretch to fill the width of the container.

<Preview className="bg-white">
  <div className="fs-md ta-ja tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-ja"
<div class="fs-md ta-ja tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Left

The text will be aligned to side of the container, which is the default alignment in most cases.

<Preview className="bg-white">
  <div className="fs-md ta-l tc-slate">
    {bleakHouse}
  </div>
</Preview>

```html "ta-l"
<div class="fs-md ta-l tc-slate">
  Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
</div>
```

## Match Parent

The text will inherit the text alignment from its parent element.

<Preview className="bg-white">
  <div className="ta-r">
    <div className="fs-md ta-mp tc-slate">
      {bleakHouse}
    </div>
  </div>
</Preview>

```html "ta-mp" "ta-r"
<div class="ta-r">
  <div class="fs-md ta-mp tc-slate">
    Michaelmas term lately over, and the Lord Chancellor sitting in Lincoln's Inn Hall...
  </div>
</div>
```
