## Installation

Install Yumma CSS and integrate it with your favorite frameworks, such as [React](/docs/guides/react), [Vue](/docs/guides/vue) or [Astro](/docs/guides/astro), in a matter of seconds.

<Tabs>
  <TabItem label="Yumma CLI" icon="forward-slash">
    <Steps>
      1. **Install Yumma CSS**

          Add `yummacss` to your project.

          <PackageManagers pkg="yummacss -D" title="Installing dependencies..." />

          <br />

      2. **Set up configuration**

          Include the paths to all source files in [`yumma.config.js`](/docs/first-steps#configuration-options) file.

          ```js title="yumma.config.js" mark={2-3}
          module.exports = {
            source: ["./src/**/*.html"],
            output: "./src/styles.css",
          };
          ```

      3. **Build styles**

          Run the `watch` command to generate styles in your CSS file.

          <PackageManagers type="exec" pkg="yummacss" args="watch" title="Building styles..." />
    </Steps>

  </TabItem>

  <TabItem label="CDN" icon="rocket">
    <Steps>
      1. **Add the CDN to your main CSS file**

          ```css title="src/styles.css" mark={1}
          @import url("https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@latest/dist/yumma.min.css");
          ```

      2. **Done!**

          You're all set to start using Yumma CSS utility classes in your project.

          ```html title="src/index.html" mark={5,8}
          <html>
            <head>
              <meta charset="UTF-8" />
              <meta name="viewport" content="width=device-width, initial-scale=1.0" />
              <link rel="stylesheet" href="/src/styles.css" />
            </head>
            <body>
              <h1 class="fs-sm fw-700 ta-c tc-indigo">Yumma CSS 🤝 HTML</h1>
            </body>
          </html>
          ```
    </Steps>
