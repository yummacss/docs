## Installation

Install Yumma CSS and integrate it with your favorite frameworks, such as [React](/docs/guides/react), [Next.js](/docs/guides/nextjs) or [Astro](/docs/guides/astro), in a matter of seconds.

<Tabs>
  <TabItem label="Node.js" icon="forward-slash">
    <Steps>
      1. **Install Yumma CSS:**

          Add `yummacss` to your project.

          ```bash title="Installing dependencies..." frame="terminal"
          npm install yummacss@latest
          ```

      2. **Include** `yummacss` **in your main CSS file:**

          ```css title="src/globals.css" mark={1}
          @import "/node_modules/yummacss/dist/yumma.min.css";
          ```

      3. **Done!**

          You're all set to start using Yumma CSS utility classes in your project.

          ```html title="src/index.html" mark={5,8}
          <html>
            <head>
              <meta charset="UTF-8" />
              <meta name="viewport" content="width=device-width, initial-scale=1.0" />
              <link rel="stylesheet" href="/src/globals.css" />
            </head>
            <body>
              <h1 class="fs-sm fw-700 ta-c tc-indigo">Yumma CSS 🤝 Node.js</h1>
            </body>
          </html>
          ```
    </Steps>

  </TabItem>
