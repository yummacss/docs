    <Steps>
      1. **Add the CDN to your main CSS file**

          ```css title="src/styles.css" mark={1}
          @import url("https://unpkg.com/yummacss@v3/dist/yumma.min.css");
          ```

      2. **Done!**

          You're all set to start using Yumma CSS utility classes in your project.

          ```html title="src/index.html" mark={4}
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="/src/styles.css" />
          </head>
          ```
