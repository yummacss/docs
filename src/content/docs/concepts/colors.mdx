## Using color shades

To specify lighter or darker shade to a given color, you can use the `[class]-l-[color]-[shade]`
selector to get a lighter color or `[class]-d-[color]-[shade]` selector to get a darker color:

<Aside title="Using shades" type="note">

The current color range is restricted to shades from 1 to 6 for both light and dark variants, with the exclusion of the base color.

```html "bg-l-cyan-3" "bg-d-cyan-3"
<!-- Light cyan -->
<button class="bg-l-cyan-3 fw-600 px-5 py-1 rad-1 tc-white">Subscribe</button>

<!-- Dark cyan -->
<button class="bg-d-cyan-3 fw-600 px-5 py-1 rad-1 tc-white">Subscribe</button>
```

</Aside>
