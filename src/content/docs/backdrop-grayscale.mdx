## Usage

### Grayscale

Use the `bf-g-*` utilities to apply a grayscale filter to the backdrop.

<div className="p-4">
  <div className="bg-silver-2 p-8 ta-c">
    <div className="bg-white/50 bf-g-1 p-4">bf-g-1</div>
  </div>
</div>

```html "bf-g-1"
<div class="bg-white/50 bf-g-1 ...">...</div>
```

### Responsiveness

Use breakpoint prefixes to target different screen sizes.

<ResponsiveVariant prefix="bf-g" />

### Hover State

Apply styles conditionally on hover using the `h:` prefix.
