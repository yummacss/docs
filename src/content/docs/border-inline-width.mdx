## Usage

Control an element's horizontal border width. This utility sets both the left & right border widths.

<Preview>
  <div className="d-f fd-r">
    <div className="d-f ai-c jc-c w-16 h-16 bc-indigo bs-s bxw-4"></div>
  </div>
</Preview>

```html "bxw-4"
<div class="d-f fd-r">
  <div class="d-f ai-c jc-c w-16 h-16 bc-indigo bs-s bxw-4"></div>
</div>
```
