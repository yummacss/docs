## Usage

### Auto

Use the `oby-a` utility to allow scroll overflow behavior on the vertical axis.

```html "oby-a"
<div class="oby-a ...">
  ...
</div>
```

### Contain

Use the `oby-c` utility to contain scroll overflow behavior on the vertical axis.

```html "oby-c"
<div class="oby-c ...">
  ...
</div>
```

### None

Use the `oby-n` utility to disable scroll overflow behavior on the vertical axis.

```html "oby-n"
<div class="oby-n ...">
  ...
</div>
```
