## Usage

```html
<p class="ff-s ...">Serif</p>
<p class="ff-m ...">UI Monospace</p>
<p class="ff-d ...">Default</p>
```

<Hint icon="heart">Yumma CSS uses the `Default` as a default font family. Learn more about it in the [docs](/docs/base-styles#document).</Hint>
