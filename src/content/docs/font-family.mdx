## Serif

If Serif is not available, it will fall back to `Cambria`, and then to a generic `serif` font.

<Preview variant="inline">
  <p className="ff-s fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
</Preview>

```html "ff-s"
<p class="ff-s fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
```

## UI Monospace

If ui-monospace is not available, it will fall back to `Consolas`, and then to a generic `monospace` font.

<Preview variant="inline">
  <p className="ff-m fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
</Preview>

```html "ff-m"
<p class="ff-m fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
```

## Default

If Default is not available, it will fall back to a generic `sans-serif` font.

<Hint icon="heart">Yumma CSS uses the `Default` as a default font family. Learn more about it in the [docs](/docs/base-styles#document).</Hint>

<Preview variant="inline">
  <p className="ff-d fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
</Preview>

```html "ff-d"
<p class="ff-d fs-lg ta-c c-slate">Sphinx of black quartz, judge my vow.</p>
```
