## Usage

### Auto

Use the `obb-a` utility to allow scroll overflow behavior to propagate on the block axis.

```html "obb-a"
<div class="obb-a ...">
  ...
</div>
```

### Contain

Use the `obb-c` utility to contain scroll overflow behavior on the block axis.

```html "obb-c"
<div class="obb-c ...">
  ...
</div>
```

### None

Use the `obb-n` utility to disable scroll overflow behavior on the block axis.

```html "obb-n"
<div class="obb-n ...">
  ...
</div>
```
