## None

> Initial value

The text will be displayed as it is, without any transformation applied.

<Preview>
  WIP
</Preview>

```html "tt-none" layout="/src/layouts/inline.astro"
<div class="bg-indigo-1 p-4 rad-1">
  <p class="bg-white fs-lg p-4 rad-1 ta-c tc-slate tt-none">Sphinx of black quartz, judge my vow.</p>
</div>
```

## Capitalize

The first letter of each word will be converted to uppercase.

<Preview>
  WIP
</Preview>

```html "tt-c" layout="/src/layouts/inline.astro"
<div class="bg-indigo-1 p-4 rad-1">
  <p class="bg-white fs-lg p-4 rad-1 ta-c tc-slate tt-c">Sphinx of black quartz, judge my vow.</p>
</div>
```

## Lowercase
