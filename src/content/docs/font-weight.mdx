## Usage

```html
<p class="fw-100 ...">Weight 100</p>
<p class="fw-200 ...">Weight 200</p>
<p class="fw-300 ...">Weight 300</p>
<p class="fw-400 ...">Weight 400</p>
<p class="fw-500 ...">Weight 500</p>
<p class="fw-600 ...">Weight 600</p>
<p class="fw-700 ...">Weight 700</p>
<p class="fw-800 ...">Weight 800</p>
<p class="fw-900 ...">Weight 900</p>
```

<Hint icon="heart">Yumma CSS uses the value `600` for all headings. Learn more about it in the [base styles docs](/docs/base-styles).</Hint>

<Hint icon="heart">Yumma CSS uses the value `700` for elements like `<b>` and `<strong>`. Learn more about it in the [base styles docs](/docs/base-styles).</Hint>
