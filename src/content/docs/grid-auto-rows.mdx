## Usage

### Auto

Size rows based on the content and available space.

<Preview variant="inline">
    <div className="o-auto">
      <div className="d-g gar-auto gtc-auto g-4">
          <div className="ai-c bg-indigo d-f jc-c p-4 c-white">Pneumonoultramicroscopicsilicovolcanoconiosis</div>
          <div className="ai-c bg-indigo-8 d-f jc-c p-4 c-indigo-5">B</div>
          <div className="ai-c bg-indigo-8 d-f jc-c p-4 c-indigo-5">C</div>
      </div>
    </div>
</Preview>

```html "gar-auto"
<div class="o-auto">
  <div class="d-g gar-auto gtc-auto g-4">
    <div class="ai-c bg-indigo d-f jc-c p-4 c-white">Pneumonoultramicroscopicsilicovolcanoconiosis</div>
    <div class="ai-c bg-indigo-8 d-f jc-c p-4 c-indigo-5">B</div>
    <div class="ai-c bg-indigo-8 d-f jc-c p-4 c-indigo-5">C</div>
  </div>
</div>
```
