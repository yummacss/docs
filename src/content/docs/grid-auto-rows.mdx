## Auto

> Initial value

The rows will size themselves based on the content and available space in the grid container.

<Preview variant="inline">
    <div className="o-auto">

          <div className="ai-c bg-indigo d-f jc-c p-4 tc-white">Pneumonoultramicroscopicsilicovolcanoconiosis</div>
          <div className="ai-c bg-indigo-8 d-f jc-c p-4 tc-indigo-5">B</div>
          <div className="ai-c bg-indigo-8 d-f jc-c p-4 tc-indigo-5">C</div>
      </div>
    </div>
</Preview>

## Min Content

The rows will size themselves to the smallest possible height that does not cause overflow.
