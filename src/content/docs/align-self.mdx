## Usage

### Auto

Align the item based on the container's default alignment properties.

<Preview>
  <div className="d-f as-auto g-4 h-24 w-full" id="area">
    <div className="d-f ai-c f-1 jc-c p-4 bg-indigo-8 c-indigo-5">A</div>
    <div className="d-f ai-c as-auto f-1 jc-c p-4 bg-indigo c-white">B</div>
    <div className="d-f ai-c f-1 jc-c p-4 bg-indigo-8 c-indigo-5">C</div>
  </div>
</Preview>

