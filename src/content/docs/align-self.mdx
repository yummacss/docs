## Usage

### Auto

Align the item based on the container's default alignment properties.

<Preview>
  <div className="as-auto d-f g-4 h-24 w-full" id="area">
    <div className="ai-c bg-indigo-8 d-f f-1 jc-c p-4 c-indigo-5">A</div>
    <div className="ai-c as-auto bg-indigo d-f f-1 jc-c p-4 c-white">B</div>
    <div className="ai-c bg-indigo-8 d-f f-1 jc-c p-4 c-indigo-5">C</div>
  </div>
</Preview>

```html "as-auto"
<div class="d-f g-4 h-24 w-full">
  <div class="ai-c bg-indigo-8 d-f f-1 jc-c p-4 c-indigo-5">A</div>

</div>
```
