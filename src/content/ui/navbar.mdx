## Accessibility

The navbar component follows standard navigation landmark practices and the [Disclosure Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/) for the mobile menu.

### Keyboard Interaction

| Key               | Action                                                           |
|-------------------|------------------------------------------------------------------|
| `Tab`             | Moves focus through navigation links and the mobile menu toggle. |
| `Enter` / `Space` | Activates focused links or toggles the mobile menu.              |
| `Escape`          | Closes the mobile menu if it is currently open.                  |


### Roles, States, and Properties

| Property             | Description                                                      |
|----------------------|------------------------------------------------------------------|
| `nav`                | Identifies the element as a navigation landmark.                 |
| `aria-label`         | Provides an accessible label for the navigation landmark.        |
| `aria-expanded`      | Indicates if the mobile menu (disclosure) is currently expanded. |
| `aria-controls`      | References the ID of the mobile menu content.                    |
| `aria-hidden="true"` | Hides decorative icons from screen readers.                      |
| `ul` / `li`          | Provides a semantic list structure for navigation items.         |
