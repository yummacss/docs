## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Free, accessible React context menu components built with Base UI for right-click actions, nested menus, and toolbar options with keyboard navigation.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="h-sm bw-0" registryId="context-menu-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="context-menu-base"
    ```
  </TabsPanel>
</Tabs>
