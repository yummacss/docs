## Example

A component that displays a user's profile picture or initials.

```tsx mark={1}
import { Avatar, AvatarFallback, AvatarImage } from "@yummacss/react";

<Avatar className="b-1">
  <AvatarImage src="https://images.unsplash.com/photo-1543610892-0b1f7e6d8ac1?w=128&h=128&dpr=2&q=80" />
  <AvatarFallback>LT</AvatarFallback>
</Avatar>
```

## API reference

### `size`

> Default value is `md`

```ts
(property) size?: "sm" | "md" | "lg" | null | undefined
```

### `shape`

> Default value is `circle`

```ts
(property) shape?: "circle" | "square" | null | undefined
```
