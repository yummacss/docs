## What is Yumma UI?

A set of ready-to-use UI components based on [Base UI](https://base-ui.com/) for accessibility and keyboard navigation, styled with atomic classes from Yumma CSS, giving you full control over the final look and feel without any hidden abstractions.

## Getting started

Yumma UI components are for pasting into your project, and you need to set up dependencies and configuration to ensure correct functioning.

### React (Vite)

If you're using Vite for your React project, follow these steps to get started.

<Stepper>
  <Step title="Create project">
    Initialize a new React project with Vite.

    ```bash
    pnpm create vite my-app --template react-ts
    ```
  </Step>

  <Step title="Install dependencies">
    Install the core dependencies for Yumma UI.

    ```bash
    pnpm add yummacss @base-ui/react motion
    ```
  </Step>

  <Step title="Initialize Yumma CSS">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Configure Yumma CSS">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={4}
    import { defineConfig } from "yummacss";

    export default defineConfig({
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/styles.css",
    });
    ```
  </Step>

  <Step title="Import styles">
    Import the generated CSS file in your root entry file.

    ```tsx title="src/main.tsx"
    import "./styles.css";
    import React from "react";
    import ReactDOM from "react-dom/client";
    import App from "./App";

    ReactDOM.createRoot(document.getElementById("root")).render(
      <React.StrictMode>
        <App />
      </React.StrictMode>
    );
    ```
  </Step>
</Stepper>

### Next.js (App Router)

The recommended way to use Yumma UI with Next.js is by using the App Router.

<Stepper>
  <Step title="Create project">
    Initialize a new Next.js project if you haven't already.

    ```bash
    pnpx create-next-app@latest my-app
    ```
  </Step>

  <Step title="Install dependencies">
    Install the core dependencies for Yumma UI.

    ```bash
    pnpm add yummacss @base-ui/react motion
    ```
  </Step>

  <Step title="Initialize Yumma CSS">
    Create a configuration file in your project.

    ```bash
    pnpm yummacss init
    ```
  </Step>

  <Step title="Configure Yumma CSS">
    Specify the locations of all your project files in the config file.

    ```js title="yumma.config.mjs" mark={4}
    import { defineConfig } from "yummacss";

    export default defineConfig({
      source: ["./src/**/*.{ts,tsx}"],
      output: "./src/styles.css",
      normalize: true,
    });
    ```
  </Step>

  <Step title="Import styles">
    Import the generated CSS file in your root layout.

    ```tsx title="src/app/layout.tsx"
    import "./styles.css";

    export default function RootLayout({ children }) {
      return (
        <html lang="en">
          <body>{children}</body>
        </html>
      );
    }
    ```
  </Step>
</Stepper>
