## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Free, accessible React preview card components built with Base UI for product showcases, article previews, and image galleries with hover effects.

### Base

A text link triggers a profile popup on hover.

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="bw-0" registryId="preview-card-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="preview-card-base"
    ```
  </TabsPanel>
</Tabs>
