## Accessibility

The textarea component follows standard HTML accessibility practices.

### Roles, States, and Properties

| Property           | Description                                                                            |
|--------------------|----------------------------------------------------------------------------------------|
| `aria-label`       | Provides an accessible name when a visible label is not present.                       |
| `aria-describedby` | References elements that provide additional information or error messages.             |
| `aria-invalid`     | Indicates that the value entered in the field does not conform to the required format. |
| `disabled`         | Prevents user interaction and indicates the field is not currently editable.           |
