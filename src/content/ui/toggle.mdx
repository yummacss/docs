## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Implement these components in your project using Yumma CSS and Base UI.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white" value="preview">
    <Preview className="bw-0" id="toggle-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registry="toggle-base"
    ```
  </TabsPanel>
</Tabs>

### Group

A group of toggle buttons for text alignment.

<Tabs defaultValue="preview">
