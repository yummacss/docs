## Installation

Implement this component in your project using Yumma CSS, Base UI, and Motion.

```bash frame="none"
pnpm add @base-ui/react motion
```

## Examples

### Basic

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white" value="preview">
    <Preview className="bw-0" id="accordion-basic" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registry="accordion-basic"
    ```
  </TabsPanel>
</Tabs>
