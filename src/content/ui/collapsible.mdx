## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Implement this component in your project using Yumma CSS and Base UI.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="h-fc bw-0" registryId="collapsible-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="collapsible-base"
    ```
  </TabsPanel>
</Tabs>
