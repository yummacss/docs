## Installation

```bash
pnpm add @base-ui/react
```

## Examples

Free, accessible React checkbox components built with Base UI for multi-select forms, terms acceptance, and nested option lists with parent-child sync.

### Base

<Tabs defaultValue="preview">

  </TabsList>
