## Example

A form control that allows users to select one or more options from a set.

```tsx mark={1}
import { Checkbox, CheckboxIndicator } from "@yummacss/react";
import { CheckIcon } from "@radix-ui/react-icons";

<div className="d-if ai-c cg-2">
  <Checkbox defaultChecked>
    <CheckboxIndicator>
      <CheckIcon />
    </CheckboxIndicator>
  </Checkbox>
  <span className="fs-sm">Enable notifications</span>
</div>
```

## Custom styling

Apply custom styles to the `<Checkbox>` component.

```tsx 'className="..."'
<Checkbox className="...">...</Checkbox>;
```

## Extend properties

Extend the `<Checkbox>` component with the Base UI API.

```tsx "{...props}"
<Checkbox {...props}>...</Checkbox>;
```
