## Core Examples

Free and accessible Menubar React components, using Base UI and Yumma CSS.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="h-xs bw-0" registryId="menubar-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="menubar-base"
    ```
  </TabsPanel>
</Tabs>
