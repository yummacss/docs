## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Free, accessible React menubar components built with Base UI for application menus, navigation bars, and grouped actions with keyboard support.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="bw-0 h-xs" registryId="menubar-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="menubar-base"
    ```
  </TabsPanel>
</Tabs>
