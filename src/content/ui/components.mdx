## Why Yumma UI?

With Yumma UI, you will get access to dozens of free production-ready components and blocks for whatever you're building. 

Our React components are powered by [Base UI](https://base-ui.com) for accessible behavior, keyboard navigation, and focus management. HTML examples are dependency-free.

---

## Installation

Yumma UI is built on Yumma CSS, so you'll want to make sure you've got the latest version of Yumma CSS installed.

Don't know how to install Yumma CSS? Check out the [installation](/docs/installation) page.

```bash title="Terminal"
npm install yummacss@latest -D
```
