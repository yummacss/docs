## What is Yumma UI?

Yumma UI is a library of professional, production-ready components designed to help you build stunning interfaces faster. Unlike traditional component libraries that come with heavy, pre-defined styles, Yumma UI uses strictly atomic classes from **Yumma CSS**, giving you full control over the final look and feel without any hidden abstractions.

Every component is built with a focus on:

- **Strictly Atomic Styling**: No semantic tokens, no complex configurations—just pure atomic CSS power.
- **Top-Tier Performance**: Minimal CSS footprint and optimized React components.
- **Developer Experience**: Components that are easy to copy, paste, and adapt to your specific needs.

## How does it handle accessibility?

Accessibility is not an afterthought in Yumma UI. We leverage **Base UI** (by the MUI team) to handle the complex logic behind accessible components:

- **WAI-ARIA Compliance**: Automatic management of ARIA attributes, focus states, and keyboard navigation.
- **Unstyled Foundation**: Base UI provides the functionality, while Yumma UI provides the professional styling, ensuring you get the best of both worlds.
- **Focus Management**: Robust focus trapping and restoration for dialogs, menus, and popovers.

## Tech Stack

Yumma UI is built on a modern foundation of industry-leading tools:

- **React v19.2**: The latest version of React for building modern user interfaces.
- **Yumma CSS v3.10**: An atomic CSS framework designed for speed and simplicity.
- **Motion v12.30**: A powerful animation library for fluid and performant transitions.
- **Base UI v1.1**: The foundation for accessible and interactive components.
