## Example

A container component that groups related content and actions.

```tsx mark={1}
import { Card, CardTitle, CardDescription } from "@yummacss/react";

<Card>
  <CardTitle>Card Title</CardTitle>
  <CardDescription>Card Description</CardDescription>
</Card>
```

## Custom styling

Apply custom styles to the `<Card>` component.

```tsx 'className="..."'
<Card className="...">...</Card>;
```

## Extend properties

Extend the `<Card>` component with native `<div>` HTML attributes.

```tsx "{...props}"
<Card {...props}>...</Card>;
```
