## Installation

```bash
pnpm add @base-ui/react motion
```

## Examples

Free, accessible React tabs components built with Base UI for tabbed content, settings panels, and multi-section layouts with animated transitions.

### Base

<Tabs defaultValue="preview">
  <TabsList>
    <TabsTab value="preview">Preview</TabsTab>
    <TabsTab value="code">Code</TabsTab>
  </TabsList>
  <TabsPanel className="bg-white bbr-sm" value="preview">
    <Preview className="bw-0" registryId="tabs-base" />
  </TabsPanel>
  <TabsPanel value="code">
    ```tsx registryId="tabs-base"
    ```
  </TabsPanel>
</Tabs>
