# [@yummacss](www.yummacss.com)

A CLI-first CSS framework for the web with abbreviated styles.

## Installing

Install `yummacss` as a dev dependency and initialize configuration

```bash
npm install yummacss -D
npx yummacss init
```

Build styles

```bash
npx yummacss build
```
