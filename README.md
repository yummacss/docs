# Yumma CSS Docs

Official documentation website for Yumma CSS.

---

## Development

To start the docs site in development mode, from the project root, run:

```bash
pnpm install
pnpm dev
```

This runs both Next.js dev server and Yumma CSS in watch mode.

## Build

```bash
pnpm build
```
