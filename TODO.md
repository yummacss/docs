# Yumma CSS 3.10 - Release Roadmap

This list tracks the essential features to be released in Yumma CSS v3.10 to support professional UI development.

## Pseudo-Classes & Variants
- [ ] `c:` (:checked)
- [ ] `d:` (:disabled)
- [ ] `e:` (:empty)
- [ ] `fc:` (:first-child)
- [ ] `i:` (:invalid) / `v:` (:valid)
- [ ] `in:` (:indeterminate)
- [ ] `lc:` (:last-child)
- [ ] `nthc:` (:nth-child)
- [ ] `r:` (:required)
- [ ] `ro:` (:read-only)

## Transitions & Animations
- [ ] **Transition Property**: `tp-*` (e.g., `tp-a`, `tp-c`, `tp-o`)
- [ ] **Transition Duration**: `td-[0-1000]` (steps of 50ms, e.g., `td-300`)
- [ ] **Transition Timing**: `ttf-*`
  - `ttf-l` (linear)
  - `ttf-ei` (ease-in)
  - `ttf-eo` (ease-out)
  - `ttf-io` (ease-in-out)
- [ ] **Transition Delay**: `trd-[0-1000]` (steps of 50ms)

## Miscellaneous
- [ ] `sr-o` (Screen Reader Only / Visually Hidden)
